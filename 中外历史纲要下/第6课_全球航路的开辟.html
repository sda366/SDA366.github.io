航海家支持国时间初中要点高中拓展

二、重点难点深度解析

重点① 新航路开辟的动因（为什么走？）

新航路开辟的动因是高考必考内容！务必从"经济—社会—直接—精神"四个层面全面掌握。

四个层次的动因：

层面内容理解要点
经济动因
（根本原因）		①商品经济发展，资本主义萌芽
②对金银财富的渴求
③开拓新市场的需求		这是最深层的推动力。欧洲商品经济→需要更多市场→需要更多金银（货币）→必须向外扩张。记住："金银+市场"是关键词。
社会动因
（直接诱因）		①《马可·波罗行纪》对东方富庶的描述
②奥斯曼帝国垄断传统商路		前者是"拉力"（东方的吸引力），后者是"推力"（旧路走不通了），两者共同推动欧洲人寻找新航线。
精神动因
（思想驱动）		①文艺复兴的人文主义（冒险进取）
②传播基督教的宗教热情		文艺复兴打破了中世纪天主教"禁欲""安分"的观念，鼓励冒险探索和个人价值实现。
客观条件
（能不能走）		①造船技术（多桅帆船）
②航海技术（罗盘、星盘、航海图）
③地理知识（地圆说、季风规律）
④王权支持（西/葡王室资助）		动因是"想走"，条件是"能走"，两者缺一不可。注意王权支持很重要——航海需要巨额资金。

重点② 新航路开辟的条件（凭什么走？）

条件从四个方面分析：

  • 技术条件：多桅帆船（速度快、抗风浪能力强）、罗盘（定向）、星盘（定位）、航海图和地理知识的积累。
  • 政治条件：西班牙和葡萄牙完成中央集权，王室有能力提供巨额资金资助航海活动。
  • 经济条件：商品经济发展积累了足够的财富，支持冒险投资。
  • 知识条件：地圆学说的传播（托勒密《地理学》被重新发现）、季风规律的了解。
为什么是葡萄牙和西班牙先行？①两国位于伊比利亚半岛，面朝大西洋，地理位置优越；②两国在收复失地运动（Reconquista）中完成统一和中央集权；③两国积累了丰富的航海经验；④两国在宗教热情驱动下，也希望向海外传播基督教。1494年《托尔德西利亚斯条约》划分了两国势力范围（教皇子午线）。

难点 动因与条件的区分

考试常见陷阱——不要把"动因"和"条件"混淆：

动因（为什么走）条件（凭什么走）
本质推动力/驱动力保障/基础
举例对金银的渴望、寻找新市场、奥斯曼阻断旧路罗盘、多桅帆船、地圆说、王室支持
答题关键词"因为……""为了……""由于……""凭借……""依靠……""利用了……"
常见混淆❌ 把"罗盘传入"说成动因❌ 把"奥斯曼阻断商路"说成条件

💡 记忆口诀：动因是"我想去"（经济需求+社会刺激），条件是"我能去"（技术+政治+知识）。两者结合，新航路开辟才成为可能。

三、课堂训练

总分：100分  |  建议用时：25分钟

一、判断题（每题2分，共10分）

1.新航路开辟的根本原因是欧洲资本主义萌芽的发展。2分

✅ 正确。14—15世纪欧洲商品经济和资本主义萌芽的发展，对金银财富的渴求以及开拓新市场的意愿，是新航路开辟的根本原因（经济动因）。

2.哥伦布首次横渡太平洋，到达了亚洲的印度。2分

❌ 错误。哥伦布横渡的是大西洋（而非太平洋），到达的是美洲巴哈马群岛等地。他本人至死都认为自己到达的是亚洲印度，因此称当地居民为"印第安人"。实际发现美洲新大陆的是他。

3.麦哲伦是第一个完成环球航行的人。2分

❌ 错误。麦哲伦在菲律宾的麦克坦岛与当地人交战时阵亡，并未完成环球航行。是他的船队在埃尔卡诺的率领下最终回到了西班牙，完成了人类首次环球航行。应说"麦哲伦船队完成了环球航行"。

4.新航路开辟的客观条件之一是航海技术的进步，如罗盘和地圆说的应用。2分

✅ 正确。罗盘（从中国经阿拉伯传入欧洲）提供了远洋航行的定向工具；地圆说为横渡大洋提供了理论依据；多桅帆船提高了远洋航行能力。这些都是新航路开辟的重要客观条件。

5.葡萄牙和西班牙之所以率先进行新航路开辟，主要是因为两国科技水平领先其他欧洲国家。2分

❌ 错误。葡萄牙和西班牙率先开辟新航路的主要原因并非科技领先，而是：①两国面朝大西洋的地理位置优越；②两国在收复失地运动中完成统一和中央集权，王权强大；③积累了丰富的航海经验；④王室愿意资助航海活动。当时欧洲各国的航海技术水平差距并不大。

三、选择题（每题3分，共30分）

6.新航路开辟的根本原因是（　）3分

  1. 奥斯曼帝国控制了传统商路
  2. 欧洲资本主义萌芽的产生和发展
  3. 《马可·波罗行纪》的广泛传播
  4. 欧洲航海技术的进步

答案：B

A项是直接原因（推动因素之一），C项是社会动因之一，D项是客观条件而非动因。根本原因（最深层的驱动力）是经济层面的——14—15世纪欧洲商品经济和资本主义萌芽的发展，对金银和市场产生迫切需求。答题时注意区分"根本原因"与"直接原因"。

7.在新航路开辟过程中，第一次完成横渡大西洋的航海家是（　）3分

  1. 迪亚士
  2. 达·伽马
  3. 哥伦布
  4. 麦哲伦

答案：C

1492年，哥伦布在西班牙王室资助下率3艘船从西班牙出发，横渡大西洋到达美洲，是人类首次横渡大西洋的壮举。迪亚士沿非洲海岸航行，达·伽马绕过非洲到印度，麦哲伦横渡的是太平洋。

8.下列各项中，属于新航路开辟的客观条件的是（　） ①地圆学说的流行 ②罗盘的应用 ③资本主义萌芽的产生 ④多桅帆船的出现3分

  1. ①②③
  2. ①②④
  3. ①③④
  4. ①②③④

答案：B

①②④均为客观条件（技术/知识保障）。③资本主义萌芽是动因（驱动力），不是条件。"动因"是"为什么走"，"条件"是"凭什么走"，两者不能混淆。

9.1494年，葡萄牙和西班牙签订《托尔德西利亚斯条约》，其主要目的是（　）3分

  1. 联合对抗奥斯曼帝国
  2. 瓜分海外殖民势力范围
  3. 共同开辟新航路
  4. 传播基督教到东方

答案：B

《托尔德西利亚斯条约》在教皇的仲裁下，以一条经线（西经46°）将世界划分为两半：东半球归葡萄牙，西半球归西班牙。两国据此瓜分了海外殖民势力范围，避免相互冲突。这是殖民扩张早期的重要事件。

10.文艺复兴对新航路开辟的影响主要体现在（　）3分

  1. 提供了航海技术和造船技术
  2. 弘扬了人文主义，鼓励冒险进取精神
  3. 提供了开辟新航路的资金
  4. 推动了地圆学说的确立

答案：B

文艺复兴的核心思想是人文主义，强调人的价值和力量，鼓励冒险进取和探索未知世界，为航海家们提供了精神动力。A、D项属于技术/科学进步，不属于文艺复兴的直接影响。C项属于经济层面的条件。

11.与迪亚士、达·伽马的航海活动不同，哥伦布的航行开辟了（　）3分

  1. 欧洲经非洲到亚洲的航路
  2. 欧洲到美洲的新航路
  3. 欧洲到印度的直达航路
  4. 环球航行的路线

答案：B

迪亚士和达·伽马走的是"向东"路线（绕过非洲到亚洲），哥伦布走的是"向西"路线（横渡大西洋到美洲）。A是达·伽马的成就，C和D均不准确。

12.奥斯曼帝国对新航路开辟的影响是（　）3分

  1. 提供了航海技术和知识支持
  2. 阻断了东西方传统商路，迫使欧洲人另寻出路
  3. 直接资助了哥伦布的航海活动
  4. 传播了地圆说等科学知识

答案：B

奥斯曼帝国1453年灭亡拜占庭帝国后，控制了东西方之间的传统商路，征收高额关税，使东方商品在欧洲价格暴涨。这迫使欧洲各国不得不寻找绕过奥斯曼帝国的新航路，是推动新航路开辟的直接动因之一。

13.下列关于四位航海家及其成就的对应，正确的是（　）3分

  1. 迪亚士——发现美洲新大陆
  2. 达·伽马——到达非洲好望角
  3. 哥伦布——横渡太平洋
  4. 麦哲伦船队——完成环球航行

答案：D

A错误——发现美洲的是哥伦布；B错误——到好望角的是迪亚士，到印度的是达·伽马；C错误——哥伦布横渡的是大西洋，横渡太平洋的是麦哲伦。

14.新航路开辟过程中，葡萄牙人和西班牙人的航海方向分别是（　）3分

  1. 都向东
  2. 都向西
  3. 葡萄牙向东，西班牙向西
  4. 葡萄牙向西，西班牙向东

答案：C

葡萄牙人主要向东航行（迪亚士→好望角→达·伽马→印度），走的是绕过非洲的路线。西班牙人主要向西航行（哥伦布→美洲→麦哲伦→环球），走的是横渡大西洋/太平洋的路线。这是高中考试中常见的区分考点。

15.新航路的开辟使世界历史发生了重大转折，这个"转折"指的是（　）3分

  1. 欧洲开始从封建社会向资本主义社会过渡
  2. 世界从分散走向整体，各地区的联系日益紧密
  3. 欧洲确立了世界殖民霸权
  4. 亚洲开始落后于欧洲

答案：B

新航路开辟最深远的历史意义是"世界从分散走向整体"——此前各文明基本是独立发展的，新航路开辟将各大洲连为一体，开始了真正的全球联系。A项是长期结果而非"转折"；C项是新航路开辟后经过长期殖民竞争才实现；D项是后来的结果。

四、材料分析题（共50分）

16.（25分）阅读下列材料，回答问题。25分

材料一：1500年左右，随着欧洲商品经济的日益发展和资本主义生产的萌芽，金银日益成为货币的主要流通手段，因而货币的需求量大增。…… Ottoman帝国控制了东西方之间的传统商路，对过往商品征收重税，使东方商品在欧洲市场上的价格猛涨。……与此同时，欧洲人相信地球是圆的，认为从欧洲向西航行也能到达东方。
材料二：迪亚士说："探险是为了给国王增添财富和荣耀。"哥伦布说："黄金是一切商品中最宝贵的，黄金是财富，谁占有黄金，谁就能获得他在世上所需的一切。"

（1）根据材料一，分析新航路开辟的经济动因和直接原因。（6分）

（2）根据材料二，概括航海家们开辟新航路的主要目的。（4分）

（3）结合所学知识，除材料中提到的因素外，新航路开辟还有哪些客观条件？（8分）

（4）综合材料和所学知识，分析为什么新航路开辟的"动因"不等同于"条件"。（7分）

（1）经济动因和直接原因（6分）：经济动因：欧洲商品经济发展和资本主义萌芽，对金银（货币）的需求量增大。直接原因：奥斯曼帝国控制传统商路，征收重税，东方商品价格暴涨，迫使欧洲人寻找新航线。

（2）目的（4分）：获取财富（黄金）、为国王增添财富和荣耀、寻找通往东方的新航路。

（3）客观条件（8分）：①造船技术进步（多桅帆船）；②航海技术（罗盘、星盘、航海图）；③地理知识（地圆说、季风规律）；④政治条件（西、葡完成中央集权，王室资助）。

（4）动因与条件的区别（7分）：动因回答"为什么走"——是推动力，是主观愿望和客观需求（如对金银的渴望、奥斯曼阻断旧路）；条件回答"凭什么走"——是保障和基础，是技术能力和外部支持（如罗盘、帆船、王室资助）。动因是"想走"的驱动，条件是"能走"的保障。只有当动因与条件同时具备时，新航路开辟才成为可能。两者性质不同，不能混淆。

17.（25分）阅读下列材料，回答问题。25分

材料：在15世纪之前，世界各大文明基本上是独立发展的。1500年前后，几条关键航路的开辟改变了一切：
时间航海家支持国航线
1487年迪亚士葡萄牙欧洲→非洲好望角
1492年哥伦布西班牙欧洲→美洲
1497—1498年达·伽马葡萄牙欧洲→非洲→印度
1519—1522年麦哲伦船队西班牙环球航行

（1）根据材料中的表格信息，分别指出葡萄牙和西班牙开辟的主要航路方向有何不同。（6分）

（2）哥伦布和麦哲伦都得到了西班牙王室的支持，结合所学知识分析王室愿意资助航海活动的原因。（7分）

（3）新航路开辟后，人类社会发生了一个怎样的根本性变化？（6分）

（4）从新航路开辟的历史中，你能获得哪些启示？（6分）

（1）航路方向不同（6分）：葡萄牙的主要方向是向东（沿非洲西海岸南下，绕过好望角到达亚洲）；西班牙的主要方向是向西（横渡大西洋到达美洲，再横渡太平洋进行环球航行）。

（2）王室资助原因（7分）：①获取海外财富（黄金、香料等），增强国力；②扩大势力范围，争夺海外殖民利益；③在与其他欧洲国家的竞争中占据优势；④传播基督教的宗教目的；⑤两国已完成中央集权，有财力支持。

（3）根本性变化（6分）：世界从相对孤立、分散的状态逐步走向紧密联系的整体。各大洲之间结束了相对隔绝的状态，开始了真正意义上的全球联系，人类历史开始进入"世界历史"阶段。

（4）启示（6分，开放性）：①勇于探索未知的精神值得学习；②科学技术的进步推动历史发展（造船、航海技术是保障）；③历史发展往往带来双面效应（新航路开辟既促进了世界联系，也开启了殖民扩张）；④经济发展需求是推动社会变革的根本动力。
