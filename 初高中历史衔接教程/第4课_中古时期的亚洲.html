二、重点难点深度解析

重点① 阿拉伯帝国——东西方文明的桥梁

"桥梁"体现在三个方面：

  • 翻译运动：8—9世纪阿拔斯王朝在巴格达设立"智慧宫"，系统翻译古希腊哲学、科学著作。后来这些典籍经西班牙传入欧洲，推动了欧洲文艺复兴。
  • 贸易枢纽：阿拉伯人垄断印度洋、地中海东西方商路，将中国的丝绸、瓷器、造纸术，印度的香料、数字，欧洲的玻璃、毛织品进行交换传播。
  • 文化创造：在吸收东西方文明基础上创造出独特的阿拉伯-伊斯兰文明：改进印度数字（"阿拉伯数字"）、创立代数学（代数一词即来自阿拉伯语al-jabr）、医学巨著《医典》长期为欧洲医学教材。
造纸术西传路线：中国（汉代发明）→ 751年怛罗斯之战被阿拉伯人俘获工匠 → 8世纪巴格达设造纸工场 → 10世纪传入埃及 → 12世纪传入西班牙 → 欧洲各地。造纸术的西传直接推动了欧洲文化传播，是文明交流改变世界进程的经典案例。

重点② 奥斯曼帝国——垄断商路与推动新航路开辟

关键逻辑链：

  • 奥斯曼帝国1453年灭亡拜占庭帝国，控制了东西方传统商路（经地中海→黑海→中亚）
  • 对过往商旅征收高额关税，使从东方运到欧洲的商品价格暴涨
  • 欧洲各国急需寻找绕过奥斯曼帝国的新航线 → 成为推动新航路开辟的直接经济动因之一
  • 这就是为什么说"奥斯曼帝国的扩张间接催生了全球化"——大历史视角的因果链条
米勒特制度：奥斯曼帝国允许被征服地区的基督教徒、犹太教徒保持自己的宗教信仰和社区自治，只需缴纳人头税（吉兹亚），不用服兵役。这种"宗教宽容"政策实际上是一种高效的统治策略——既减少了统治阻力，又增加了财政收入。与欧洲中世纪宗教迫害形成鲜明对比。

重点③ 日本——从"学唐"到"武士掌权"

日本中古史的核心线索：

  • 646年大化改新：模仿唐朝中央集权制度（班田收授法、租庸调制、律令制），使日本从奴隶社会过渡到封建社会。
  • 10世纪后庄园制度兴起：班田制瓦解，土地私有化，地方豪族建立私人武装（武士）。
  • 12世纪末镰仓幕府建立：日本进入"幕府政治"时代——天皇仅是名义上的首脑，实权掌握在幕府将军（武士首领）手中。
  • 武士道精神：忠诚、勇敢、信义、节俭，对日本文化影响深远。

难点 亚洲中古时期区域文明的多元面貌

多元面貌的成因分析：

地区自然环境历史文化背景独特文明特征
西亚（阿拉伯）沙漠绿洲为主，地处三大洲交通要冲多民族交汇，一神教传统政教合一、商业文明、东西方桥梁
小亚/巴尔干（奥斯曼）地跨欧亚的半岛，控制海峡继承拜占庭传统，伊斯兰化军事扩张型帝国、米勒特多元共存
东亚（日本）岛国，海洋阻隔深受中国影响但保持独立性天皇+幕府双轨、武士道、和风文化
东亚（朝鲜）半岛，与中国陆路相连长期受中国文化直接影响儒家治国、科举制、"事大"外交
南亚（印度）次大陆，相对封闭又可进入印度教传统+伊斯兰教传入宗教多元化、种姓制度延续
东南亚热带季风，海上通道印度文化传入+本土融合佛教/印度教王国、吴哥文明

💡 理解方法：自然环境决定了经济方式，经济方式影响了政治制度和社会结构，历史文化背景塑造了独特的文明面貌。六大地区虽处于同一时代，但因上述差异而呈现出丰富的多元面貌——这正是世界历史的魅力所在。

三、课堂训练

总分：100分  |  建议用时：25分钟

一、判断题（每题2分，共10分）

1.阿拉伯帝国的哈里发既是政治首脑也是宗教领袖，实行的是政教合一的制度。2分

✅ 正确。哈里发（"继任者"之意）是穆罕默德死后伊斯兰教政教合一的最高领袖，集宗教权威与政治权力于一身。

2.日本大化改新完全照搬了唐朝的制度，没有任何本土特色。2分

❌ 错误。大化改新虽然深受唐朝影响，但并非"完全照搬"。日本在模仿唐朝制度的同时，保留了天皇制的神统观念，后期演变为独特的幕府政治和庄园制度，形成了"唐风+本土"的和风文化。

3.奥斯曼帝国垄断东西方商路后，欧洲人被迫寻找新航路，这直接推动了地理大发现。2分

✅ 正确。1453年奥斯曼帝国攻占君士坦丁堡后，控制了传统的东西方商路，征收高额关税，使东方商品在欧洲价格暴涨。这是推动欧洲人寻找新航路的重要经济动因之一。

4.中古时期朝鲜深受中国影响，完全采用中国科举制的模式，没有任何区别。2分

❌ 错误。朝鲜虽然仿效中国设立科举制，但有自己的特点。高丽时期主要考儒学经典，后来李朝时期科举内容也与朝鲜实际相结合。朝鲜在吸收中国制度的同时，也进行了本土化改造。

5.阿拉伯数字是阿拉伯人从零发明的数字体系。2分

❌ 错误。"阿拉伯数字"实际上起源于古印度。古印度人创造了0到9的计数法，阿拉伯人将其改造并传入欧洲，因此欧洲人称之为"阿拉伯数字"。这是文明交流与传播的典型案例。

三、选择题（每题3分，共30分）

6.阿拉伯帝国在中古世界历史中被称为"东西方文明的桥梁"，主要是因为（　）3分

  1. 阿拉伯帝国是当时世界上面积最大的帝国
  2. 阿拉伯人创造了辉煌的伊斯兰文明
  3. 阿拉伯人在东西方文化交流中起到承上启下的中介作用
  4. 阿拉伯帝国征服了欧亚非三大洲

答案：C

阿拉伯帝国地处东西方交通要冲，通过翻译运动保存和传播了古希腊文化，同时将中国的造纸术、印度的数字等传播到欧洲，在文明交流中发挥了不可替代的桥梁作用。

7.1453年，奥斯曼帝国攻占君士坦丁堡，灭亡了（　）3分

  1. 罗马帝国
  2. 阿拉伯帝国
  3. 拜占庭帝国（东罗马帝国）
  4. 波斯帝国

答案：C

1453年奥斯曼帝国苏丹穆罕默德二世攻陷君士坦丁堡，延续了千年的拜占庭帝国（东罗马帝国，395—1453）灭亡，标志着欧洲中世纪的结束。

8.日本大化改新的最主要影响是（　）3分

  1. 使日本确立了武士政治
  2. 使日本从奴隶社会过渡到封建社会
  3. 使日本实现了工业化
  4. 使日本走上了对外扩张道路

答案：B

646年大化改新仿效中国隋唐制度，废除贵族世袭制，实行班田收授法和租庸调制，建立了中央集权国家体制，标志着日本从奴隶社会向封建社会的过渡。

9.奥斯曼帝国的扩张对欧洲产生的深远影响不包括（　）3分

  1. 垄断东西方商路，使商品价格暴涨
  2. 推动欧洲人寻找新航路
  3. 直接导致欧洲文艺复兴
  4. 灭亡拜占庭帝国

答案：C

文艺复兴的兴起原因是多方面的（资本主义萌芽、古希腊罗马典籍的重新发现、城市兴起等），不能说奥斯曼帝国的扩张"直接导致"了文艺复兴。奥斯曼帝国的影响主要体现在经济层面（垄断商路→推动新航路开辟）和政治层面（灭亡拜占庭）。

10.下列关于中古时期朝鲜的表述，正确的是（　）3分

  1. 7世纪新罗统一朝鲜半岛后独立发展，未受中国影响
  2. 10世纪高丽王朝实行田柴科制度
  3. 朝鲜自古以来就是中国的附属国
  4. 朝鲜没有实行科举制度

答案：B

10世纪高丽王朝建立后实行田柴科制度（国家按等级分配土地），同时仿照唐朝设立科举。A项错误——朝鲜深受中国影响；C项过于绝对化——朝鲜与中国有宗藩关系但并非"自古以来"的附属国；D项错误——朝鲜仿效中国实行了科举制度。

11.日本幕府政治的本质特征是（　）3分

  1. 天皇掌握最高实权
  2. 武士集团掌握国家实权，天皇成为名义首脑
  3. 议会民主政治
  4. 贵族共和制度

答案：B

12世纪末镰仓幕府建立后，日本进入幕府政治时代。天皇仅是名义上的国家首脑，实际权力掌握在幕府将军（武士首领）手中。这种"天皇+幕府"双轨体制延续了约700年。

12."阿拉伯数字"实际上起源于（　）3分

  1. 阿拉伯半岛
  2. 古埃及
  3. 古印度
  4. 古巴比伦

答案：C

0—9的计数法起源于古印度，阿拉伯人学会后加以改进，并传入欧洲，因此欧洲人称之为"阿拉伯数字"。这一名称反映了文明交流传播的路径。

13.阿拉伯帝国阿拔斯王朝时期在巴格达设立的"智慧宫"，其主要功能是（　）3分

  1. 训练军队
  2. 翻译和研究古希腊、印度等地区的学术著作
  3. 进行宗教审判
  4. 管理帝国财政

答案：B

"智慧宫"是阿拔斯王朝在巴格达设立的国家学术机构，集中了大量学者从事翻译工作，将古希腊哲学、科学著作和印度数学、医学典籍翻译为阿拉伯语，对保存和传播古典文化做出了巨大贡献。

14.中古时期，东南亚地区深受哪种外来文化的影响？（　）3分

  1. 伊斯兰教文化
  2. 基督教文化
  3. 印度文化（印度教和佛教）
  4. 日本文化

答案：C

中古时期东南亚深受印度文化影响，印度教和佛教通过海上贸易传入东南亚地区，出现了如柬埔寨的吴哥窟（印度教-佛教混合）等代表性文明成果。

15.下列关于中古时期亚洲文明的表述，最能体现"文明的多元性"的是（　）3分

  1. 亚洲各地都出现了强大的封建帝国
  2. 亚洲各地都信仰伊斯兰教
  3. 亚洲各地区因自然环境和历史传统的不同，形成了各具特色的文明
  4. 亚洲各地都仿效中国建立了中央集权制度

答案：C

中古时期亚洲的阿拉伯帝国、奥斯曼帝国、日本、朝鲜、印度、东南亚等地区，因自然环境、历史文化背景和社会经济发展的不同，形成了各具特色的文明面貌，这正是文明多元性的体现。B、D项明显错误；A项也不准确——并非所有地区都出现了强大的封建帝国。

四、材料分析题（共50分）

16.（25分）阅读下列材料，回答问题。25分

材料一：阿拉伯人善于学习吸收其他民族的优秀文化成果。他们在巴格达设立"智慧宫"，大量翻译古希腊的哲学、科学著作，以及印度的数学、医学典籍……他们又把自己改进的数字体系、医学成就等传播到欧洲，对世界文化发展做出了重大贡献。
材料二：1453年，奥斯曼帝国攻陷君士坦丁堡，控制了东西方之间的传统商路。对过往商旅征收高额关税，使得从东方运到欧洲的商品价格暴涨。这促使欧洲各国急切地想要寻找通往东方的新航线。

（1）根据材料一，概括阿拉伯人在文化交流中扮演了什么角色。（6分）

（2）材料一中提到的"改进的数字体系"是什么？它实际上起源于哪里？（6分）

（3）根据材料二，分析奥斯曼帝国控制商路对欧洲产生了什么影响。（7分）

（4）综合两则材料，说明中古时期亚洲帝国对世界历史发展产生了哪些重要影响。（6分）

（1）角色（6分）：阿拉伯人在东西方文化交流中扮演了"桥梁"和"中介"的角色。他们通过翻译运动吸收和保存了古希腊、印度等地区的文化成果，同时又将这些文化成果传播到欧洲，促进了不同文明之间的交流与融合。

（2）数字体系（6分）："改进的数字体系"即"阿拉伯数字"（0—9的十位计数法），实际上起源于古印度。阿拉伯人学习后加以改进，并经西班牙传入欧洲，成为世界通用的数字体系。

（3）影响（7分）：①经济上：垄断东西方商路，征收高额关税，使东方商品在欧洲价格暴涨；②政治/历史上：迫使欧洲各国寻找通往东方的新航线，成为推动新航路开辟（地理大发现）的直接经济动因之一；③客观上推动了世界从分散走向整体的历史进程。

（4）综合影响（6分）：中古时期亚洲帝国对世界历史的影响主要体现在：①阿拉伯帝国保存和传播了古典文化，促进了东西方文明的交流与融合；②奥斯曼帝国的扩张改变了东西方贸易格局，间接推动了新航路开辟和全球化进程；③亚洲各帝国的兴衰反映了不同文明的发展模式，丰富了世界文明的多元面貌。

17.（25分）阅读下列材料，回答问题。25分

材料一：646年，日本孝德天皇颁布改新诏书，仿效中国唐朝制度，废除贵族世袭制，实行班田收授法和租庸调制，建立以天皇为中心的中央集权制度。
材料二：10世纪以后，随着班田制的瓦解，日本各地庄园林立。庄园主为了保护自己的利益，开始蓄养私人武装（武士）。12世纪末，源赖朝在镰仓建立幕府，日本进入以武士为核心的幕府政治时代。

（1）日本大化改新的主要学习对象是谁？这次改革使日本社会发生了什么根本性变化？（6分）

（2）根据材料二，说明日本幕府政治是怎样产生的。（7分）

（3）比较日本大化改新前后的政治体制有什么不同。（6分）

（4）日本中古时期的政治演变对中国有什么启示？（6分）

（1）学习对象与变化（6分）：日本大化改新主要仿效中国唐朝的制度进行改革。这次改革使日本从奴隶社会过渡到封建社会，建立了以天皇为中心的中央集权国家体制。

（2）幕府政治的产生（7分）：①班田制逐渐瓦解，土地私有化发展，各地庄园兴起；②庄园主为保护自身利益，蓄养武士作为私人武装；③武士集团势力壮大，12世纪末源赖朝在镰仓建立幕府，标志着武士掌握了国家实权，天皇成为名义首脑，日本进入幕府政治时代。

（3）政治体制比较（6分）：改新前：贵族世袭制，地方豪强割据，中央权力薄弱。改新后（大化改新）：以天皇为中心的中央集权制，废除贵族世袭，实行班田收授法。幕府时期：天皇仅保留名义地位，实权掌握在幕府将军（武士首领）手中，形成"天皇+幕府"双轨体制。

（4）启示（6分，开放性试题）：①一个国家要善于学习吸收外来先进文明，但同时要根据本国实际进行创新和改造（如日本的"唐风+本土"）；②政治制度必须适应社会经济发展的变化（班田制→庄园制→幕府制）；③改革是推动社会进步的重要动力；④保持开放心态是文明发展的必要条件。
