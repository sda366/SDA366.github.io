二、重点难点深度解析

重点① 西非三大帝国的兴衰更替

一条商路、三大帝国、一个贸易逻辑：

  • 贸易核心：撒哈拉以南的西非盛产黄金，但不产盐；北非有盐但缺黄金。黄金-盐贸易成为西非经济的命脉。
  • 加纳（8—13世纪）：最早控制盐金贸易，称"黄金之国"。11世纪后受阿尔摩拉维德王朝入侵而衰落。
  • 马里（13—15世纪）：取代加纳，版图更大。曼萨·穆萨国王1324年朝圣麦加，沿途散发大量黄金，使埃及金价暴跌十年，震动整个伊斯兰世界，使马里名扬天下。
  • 桑海（15—16世纪）：取代马里成为西非最大帝国。阿斯基亚大帝推行伊斯兰化改革，建立官僚体制。廷巴克图桑科尔大学汇聚各地学者，是非洲的学术文化中心。
廷巴克图——非洲的"学术之城"：位于今马里共和国，曾是西非最重要的贸易与学术中心。15—16世纪鼎盛时期，拥有近200所古兰经学校和桑科尔大学，藏书达数十万册，涵盖神学、法学、天文学、医学等领域，吸引来自北非和中东的学者前来求学。廷巴克图的手抄本至今仍有数十万册保存于当地图书馆。

重点② 美洲三大文明的辉煌成就

在没有与欧亚大陆交流的情况下，美洲文明独立创造了令人惊叹的成就：

  • 玛雅——"美洲的希腊"：
    • 精密历法：260天宗教历+365天太阳历，精确度超过同时期欧洲使用的儒略历
    • 数学：独立发明"零"的概念（与印度几乎同时），20进制
    • 文字：独特的象形文字体系，是美洲唯一完整发展出文字的文明
    • 建筑：阶梯式金字塔（蒂卡尔金字塔高70米）、天文观象台
  • 阿兹特克——"湖上的帝国"：
    • 特诺奇蒂特兰：建在特斯科科湖中岛上的城市，人口达20万，比同期巴黎和伦敦还大
    • 浮动园地（奇南帕）：湖中人工岛农业，一年可收获数季
    • 大规模建筑：大型神庙、宫殿、广场
  • 印加——"安第斯的帝国"：
    • 版图：南美洲最大帝国，沿安第斯山脉绵延4000多公里
    • 道路系统：总长超过3万公里，被誉为"印加的万里长城"
    • 奇普（结绳记事）：用绳结的颜色、位置、粗细记录信息，是独特的记事方式
    • 建筑：马丘比丘——建在海拔2430米山脊上的城市，石块间不用任何粘合剂却严丝合缝

难点 与欧亚文明比较——独立发展的多元文明

核心理解：非洲和美洲文明证明了"多元独立起源"是人类文明的常态。

特征非洲文明美洲文明
独立起源✅ 人类最早发源地，文明独立发展✅ 与欧亚隔绝约1.5万年，完全独立发展
共同成就农业、冶金、城市、贸易农业（玉米）、城市、历法、建筑
与欧亚差异无独立发明文字体系（部分受伊斯兰教影响后使用阿拉伯语），但口头文学丰富玛雅独立发明文字和"零"；但未发明轮式交通工具、铁器和火药
交流状况跨撒哈拉商路+印度洋贸易，与外部有交流长期隔绝，与欧亚大陆几乎无交流
文明中断15世纪起大西洋奴隶贸易、殖民入侵严重冲击16世纪被西班牙征服，文明断裂

💡 关键结论：

  • 文明不是某个地区的"专利"，人类各群体都有能力创造灿烂文明
  • 隔绝条件下的文明独立发展，证明人类智慧具有普遍性
  • 但隔绝也导致技术积累受限（如美洲无轮子、无铁器），说明文明交流对发展至关重要
  • 美洲三大文明的毁灭，揭示了殖民扩张对人类文明的巨大破坏

三、课堂训练

总分：100分  |  建议用时：25分钟

一、判断题（每题2分，共10分）

1.古代非洲与古代美洲的文明都是独立发展起来的，没有受到欧亚大陆文明的直接影响。2分

✅ 基本正确。美洲文明在与欧亚大陆长期隔绝的情况下完全独立发展。古代非洲文明虽然通过与北非和阿拉伯世界的贸易受到伊斯兰教等影响，但其核心文明成果（农业、冶金、社会制度等）是独立创造的。

2.玛雅人是美洲唯一发明了文字体系的文明。2分

✅ 正确。玛雅文明独立发展出了完整的象形文字体系，是美洲三大文明中唯一拥有完整文字的。阿兹特克使用的是图画文字（不完全的符号系统），印加使用结绳记事（奇普），都不是完整的文字体系。

3.印加人发明了轮式交通工具用于交通运输。2分

❌ 错误。印加人虽然修建了超过3万公里的道路系统，但并没有发明轮式交通工具。美洲文明由于长期与欧亚隔绝，未独立发明轮子、铁器和火药。

4.西非的加纳、马里、桑海三大帝国都位于撒哈拉以南地区，以黄金-盐贸易为核心经济。2分

✅ 正确。三大帝国均位于西非萨赫勒地区（撒哈拉沙漠南缘），控制着横跨撒哈拉的黄金-盐贸易路线。南边盛产黄金、北边有盐矿，这种贸易互补性是三大帝国兴起的经济基础。

5.印第安人是从非洲迁移到美洲大陆的。2分

❌ 错误。根据主流学术观点，印第安人约1.5万年前从亚洲东北部（西伯利亚）通过白令陆桥（今白令海峡）进入美洲大陆，并非来自非洲。

三、选择题（每题3分，共30分）

6.古代西非三大帝国（加纳、马里、桑海）兴衰更替的经济基础是（　）3分

  1. 丝绸贸易
  2. 香料贸易
  3. 黄金-盐贸易
  4. 奴隶贸易

答案：C

西非盛产黄金但不产盐，北非有盐但缺黄金，这种互补性催生了横跨撒哈拉的黄金-盐贸易，成为三大帝国兴起的经济命脉。奴隶贸易是16世纪以后才大规模发展。

7.玛雅文明在数学方面的突出成就是（　）3分

  1. 发明了十进制
  2. 独立发明了"零"的概念
  3. 发明了几何学
  4. 发明了代数学

答案：B

玛雅人独立发明了20进制计数法和"零"的概念，这一成就与古印度人几乎同时。A项错误——玛雅使用的是20进制而非十进制。C、D项是古希腊/阿拉伯文明的成就。

8.马里国王曼萨·穆萨朝圣麦加时沿途散发大量黄金，这一事件的直接影响是（　）3分

  1. 使马里帝国的黄金储备耗尽
  2. 使马里帝国在伊斯兰世界声名远扬
  3. 引发了欧洲的十字军东征
  4. 促使葡萄牙人开辟新航路

答案：B

1324年曼萨·穆萨朝圣途经开罗散发大量黄金，使当地金价暴跌十年，震动整个伊斯兰世界，使马里帝国声名远扬，同时也促进了伊斯兰文化在西非的传播。A项不符合史实——马里的黄金储备非常丰富。

9.下列关于印加帝国的表述，不正确的是（　）3分

  1. 建立了完善的道路系统
  2. 使用结绳记事
  3. 发明了完整的文字体系
  4. 首都设在库斯科

答案：C

印加人没有发明完整的文字体系，而是使用"奇普"（结绳记事）来记录信息。美洲三大文明中，只有玛雅文明拥有完整的文字体系。

10.导致阿兹特克帝国和印加帝国灭亡的共同原因是（　）3分

  1. 内部阶级矛盾激化
  2. 自然灾害
  3. 西班牙殖民者的入侵
  4. 伊斯兰教的传播

答案：C

1519年科尔特斯率领西班牙军队征服阿兹特克帝国，1532年皮萨罗率领西班牙军队征服印加帝国。两个美洲帝国都是被西班牙殖民者入侵而灭亡的。西班牙人利用军事技术优势（火器、骑兵）和疾病（天花）击败了美洲文明。

11.古代非洲文明与古代美洲文明的共同点是（　） ①都是独立起源的文明 ②都创造了辉煌的城市文明 ③都曾与欧亚大陆进行广泛交流 ④都受到了外来文明的深刻影响3分

  1. ①②
  2. ①②④
  3. ①③④
  4. ①②③④

答案：A

①正确——两大地区的文明都是独立起源的。②正确——非洲有廷巴克图等城市，美洲有特诺奇蒂特兰等城市。③错误——美洲文明长期与欧亚隔绝，没有进行"广泛交流"。④不完全正确——非洲部分地区受伊斯兰教影响，但美洲在与欧洲接触前几乎不受外来文明影响。

12.古代美洲三大文明都崇拜太阳神，这反映了（　）3分

  1. 美洲文明之间存在密切的文化交流
  2. 农业文明对自然的共同依赖
  3. 美洲文明都起源于同一地区
  4. 太阳神崇拜是世界上最古老的宗教

答案：B

美洲三大文明都是以农业为基础的社会，太阳对农业生产至关重要（光照、温度、季节变化），因此都发展出了太阳崇拜。这反映了不同文明在面对相同自然条件时可能产生相似的文化创造，而非相互交流的结果。

13.非洲的廷巴克图在15—16世纪成为重要学术中心，主要原因是（　）3分

  1. 欧洲殖民者建立了大学
  2. 桑海帝国和伊斯兰文化的推动
  3. 基督教传播的结果
  4. 工业革命的影响

答案：B

廷巴克图是桑海帝国时期西非的文化中心。桑海帝国的阿斯基亚大帝推行伊斯兰化改革，廷巴克图汇聚了大量伊斯兰学者，拥有桑科尔大学等近200所宗教学校，成为非洲的学术文化中心。

14.印加帝国修建的超过3万公里的道路系统，其主要功能不包括（　）3分

  1. 加强中央对地方的控制
  2. 便于军队快速调动
  3. 促进帝国内部的物资交流
  4. 用于与欧洲进行海外贸易

答案：D

印加帝国的道路系统主要用于加强中央集权统治：便于传达政令、调动军队、运输物资和传递信息（使用"查斯基"接力信使系统）。印加帝国与欧亚大陆完全隔绝，不存在与欧洲的海外贸易。

15.下列最能体现美洲文明"独立发展"特征的是（　）3分

  1. 玛雅的象形文字与古埃及象形文字相似
  2. 阿兹特克的金字塔与埃及金字塔外形相似
  3. 玛雅独立发明了"零"的概念和20进制
  4. 印加的道路系统与中国长城类似

答案：C

"零"的概念是高度抽象的数学成就，玛雅人在与外界完全隔绝的情况下独立发明了这一概念，最能体现"独立发展"的特征。A、B、D项虽然表面相似，但金字塔形状、文字形式等可能是人类面对类似需求时的趋同现象，不能作为"独立发展"的最有力证据。

四、材料分析题（共50分）

16.（25分）阅读下列材料，回答问题。25分

材料一：廷巴克图位于撒哈拉沙漠南缘，是西非最重要的贸易与学术中心。在15—16世纪鼎盛时期，城中拥有近200所古兰经学校和桑科尔大学，藏书达数十万册，吸引来自北非和中东的学者前来求学。来自摩洛哥的旅行者利奥·阿非利加努斯写道："这里出售的手抄本比任何其他商品都更有利可图。"
材料二：古代美洲居民在与欧亚大陆长期隔绝的情况下，独立创造了辉煌的文明。玛雅人发明了精密的历法和完整的文字系统；阿兹特克人建造了人口超过20万的特诺奇蒂特兰城；印加人修建了超过3万公里的道路网络。然而，1492年哥伦布到达美洲后，这些文明在短短数十年内被西班牙殖民者摧毁。

（1）根据材料一，概括廷巴克图在15—16世纪的特点。（6分）

（2）结合所学知识，分析廷巴克图成为西非学术中心的原因。（6分）

（3）根据材料二，说明美洲三大文明的共同特点。（6分）

（4）结合材料和所学知识，分析美洲三大文明迅速灭亡的原因。（7分）

（1）特点（6分）：①是重要的贸易城市（位于撒哈拉商路要冲）；②是学术文化中心（拥有大量学校和大学、丰富的藏书）；③具有国际化特征（吸引来自北非和中东的学者）。

（2）原因（6分）：①桑海帝国的支持（阿斯基亚大帝推行伊斯兰化改革）；②地处黄金-盐贸易要冲，经济繁荣为文化发展提供物质基础；③伊斯兰教传播推动了教育的发展；④跨撒哈拉商路促进了文化学术交流。

（3）共同特点（6分）：①独立发展（与欧亚大陆隔绝）；②以农业为基础；③创造了辉煌的城市文明；④有独特的科技成果（历法、文字、建筑等）；⑤16世纪均被西班牙殖民者征服而灭亡。

（4）灭亡原因（7分）：①军事技术悬殊——西班牙人有火器、钢铁盔甲、骑兵和战船，美洲人只有石器、木器武器；②疾病传播——西班牙人带来了天花、麻疹等传染病，美洲人毫无免疫力，人口大量死亡（据估计阿兹特克人口减少80%以上）；③内部分裂——西班牙人利用美洲帝国与其他部落之间的矛盾，拉拢盟友进行征服；④美洲文明长期隔绝发展，缺乏应对外来威胁的经验。

17.（25分）阅读下列材料，回答问题。25分

材料一：在古代非洲，黄金-盐贸易是撒哈拉以南地区最重要的经济活动。西非盛产黄金但不产盐，食盐对当地居民来说是必需品。北非有丰富的盐矿但缺乏黄金。这种互补性催生了横跨撒哈拉的庞大贸易网络。
材料二：古代非洲和美洲的文明发展证明，人类各个群体都有能力创造灿烂文明。但同时，由于地理隔绝，这些文明在科技发展方面也受到了限制。美洲文明直到16世纪仍未发明轮式交通工具、铁器和火药。

（1）根据材料一，分析西非黄金-盐贸易网络得以形成的原因。（6分）

（2）结合所学知识，说明黄金-盐贸易如何推动了西非三大帝国的兴衰更替。（7分）

（3）根据材料二，概括地理隔绝对文明发展的双重影响。（6分）

（4）综合上述材料，谈谈你对"文明的多元性"的认识。（6分）

（1）原因（6分）：①地理上的互补性——西非产黄金不产盐，北非有盐缺黄金；②需求刚性——盐是生存必需品，黄金是贵重货币和装饰品，双方都有强烈需求；③商路贯通——跨撒哈拉商路连接了南北方，促进了贸易交流。

（2）推动作用（7分）：①贸易带来了大量财富，为帝国提供了经济基础；②贸易城市（如廷巴克图）发展成为政治和文化中心；③贸易促进了伊斯兰教的传播和文化交流；④控制商路的能力决定帝国的兴衰——加纳最早控制商路兴起，失去控制后衰落，马里和桑海相继取代。

（3）双重影响（6分）：正面影响：隔绝环境使文明独立发展，创造出独特灿烂的文明成果（如玛雅的"零"和象形文字）。负面影响：隔绝限制了技术和知识的交流积累，导致某些关键技术（轮子、铁器、火药等）未能独立发明或传播，使文明在面对外部冲击时处于弱势。

（4）认识（6分，开放性）：①多元性是人类文明的常态——不同地区、不同环境下都能创造出独特的文明；②文明交流对发展至关重要——隔绝虽然能孕育独特性，但也限制了进步的空间；③尊重每一种文明的独特价值——美洲和非洲文明证明人类智慧具有普遍性；④殖民扩张对多元文明造成了巨大破坏，保护文明多样性是人类的重要课题。
