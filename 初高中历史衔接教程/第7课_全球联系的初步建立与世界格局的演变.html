二、重点难点深度解析

重点① "哥伦布大交换"及其影响

"哥伦布大交换"是近年高考新热点！理解物种交换的双向性和深远影响。
交换方向物种/物品影响
美洲→旧大陆玉米、马铃薯、番薯（甘薯）、番茄、辣椒、花生、南瓜、可可、烟草极大增加了旧大陆粮食产量；马铃薯推动欧洲人口增长；玉米和番薯传入中国缓解粮食压力
旧大陆→美洲小麦、水稻、大麦、燕麦
马、牛、羊、猪、鸡
葡萄、橄榄、香蕉		改变了美洲的农业和饮食结构；马和牛使印第安人获得新的交通工具和耕作工具
旧大陆→美洲（负面的）天花、麻疹、流感、斑疹伤寒等传染病美洲原住民缺乏免疫力，人口骤减80%以上（约从5000万降至不足1000万）
一个容易被忽略的考点——马铃薯如何改变世界？马铃薯原产美洲，传入欧洲后因产量高、适应性强，迅速成为欧洲底层人民的主食。18世纪马铃薯广泛种植直接推动了欧洲人口的快速增长，为工业革命提供了充足的劳动力。在中国，玉米和番薯的传入使山区和贫瘠土地得以开垦，推动了明清时期的人口增长。这就是"物种交换改变世界格局"的典型案例。

重点② "三角贸易"与资本原始积累

"三角贸易"路线：

  • 出程：欧洲 → 非洲（携带工业品、火器、朗姆酒，交换黑人奴隶）
  • 中程：非洲 → 美洲（运送黑人奴隶到种植园劳动）——最残酷的一段，死亡率极高
  • 归程：美洲 → 欧洲（将种植园的蔗糖、烟草、棉花、金银运回欧洲销售）

对世界格局的影响：

  • 非洲：人口大量流失（约1200万人被贩卖），社会结构遭到破坏，长期制约非洲发展
  • 美洲：黑人成为美洲主要劳动力来源之一，深刻影响了美洲的人口构成和文化
  • 欧洲：获得了巨额利润，完成了资本原始积累，推动了资本主义发展和工业革命
答题模板——殖民扩张对欧洲的双重影响：①经济上：获得原料产地和商品市场，完成资本原始积累；②政治上：增强了欧洲各国的国力和国际地位；③客观上：推动了世界市场的形成，加速了世界从分散走向整体的进程。

重点③ 早期殖民扩张与世界格局变化

核心结论：新航路开辟后，世界格局发生了根本性变化——

  • 欧洲中心地位确立：从16世纪起，欧洲逐渐成为世界的经济和政治中心，从"边缘"走向"中心"。
  • 殖民体系雏形出现：葡萄牙→西班牙→荷兰→英国，殖民霸权不断更替，英国最终成为最大殖民帝国（"日不落帝国"）。
  • 世界从分散走向整体：各大洲通过贸易、移民、殖民活动连为一体，真正意义上的世界历史开始。
  • 资本主义经济体系形成：以西欧为中心的资本主义世界经济体系开始形成——欧洲是"中心"，美洲和非洲是"边缘"。

难点 新航路开辟对世界格局的多层次影响

从不同角度分析影响——这是高考材料题的核心能力：

角度对欧洲对世界其他地区
经济资本原始积累、世界市场雏形、价格革命美洲：种植园经济；非洲：人口流失；亚洲：被动卷入贸易网络
政治殖民霸权竞争，英国崛起为"日不落帝国"美洲原住民文明被毁灭；非洲国家衰落
人口大量欧洲人移民海外美洲：原住民锐减+非洲黑人被迫迁移+欧洲移民，人口结构巨变
文化视野开阔，推动了科学发展各文明间接触加剧（基督教传播、语言文化交融）
生态获得新物种（马铃薯、玉米等）"哥伦布大交换"改变全球生态系统

💡 答题技巧：分析"影响"类题目时，养成从"经济—政治—人口/社会—文化—生态"多角度思考的习惯，答案会更全面、得分更高。

三、课堂训练

总分：100分  |  建议用时：25分钟

一、判断题（每题2分，共10分）

1."哥伦布大交换"是指美洲与旧大陆之间物种的双向交流。2分

✅ 正确。"哥伦布大交换"是双向的：美洲的玉米、马铃薯等传入旧大陆；旧大陆的马、牛、小麦等传入美洲。同时，传染病也从旧大陆传入美洲，造成灾难性后果。

2."三角贸易"的三个角分别是欧洲、亚洲和美洲。2分

❌ 错误。"三角贸易"的三个角是欧洲、非洲和美洲（不是亚洲）。路线是：欧洲（工业品）→非洲（交换奴隶）→美洲（奴隶劳动）→ 欧洲（原料和金银）。亚洲虽然也有贸易，但不属于"三角贸易"的范畴。

3.新航路开辟后，欧洲逐渐确立了世界经济的中心地位。2分

✅ 正确。新航路开辟使欧洲从亚欧大陆的"边缘"逐渐走向世界经济的"中心"，通过殖民掠夺和全球贸易积累了巨额财富，最终推动了工业革命。

4.荷兰东印度公司是世界上第一个跨国股份制公司。2分

✅ 正确。荷兰东印度公司（VOC）成立于1602年，是世界上第一个跨国股份制公司，拥有发行股票、征税、宣战、缔约等权力，是荷兰殖民扩张的重要工具。

5.新航路开辟对美洲原住民的影响主要是推动了他们的社会进步。2分

❌ 错误。新航路开辟对美洲原住民的影响主要是灾难性的：欧洲人带来的传染病（天花等）导致美洲原住民人口骤减约80%；欧洲殖民者征服并摧毁了阿兹特克和印加等原住民文明；大量原住民被迫在种植园劳动。虽有马、小麦等新物种传入，但整体影响是负面的。

三、选择题（每题3分，共30分）

6."哥伦布大交换"中，从美洲传入旧大陆的物种不包括（　）3分

  1. 玉米
  2. 马铃薯
  3. 小麦
  4. 番茄

答案：C

小麦原产于旧大陆（西亚），是从旧大陆传入美洲的物种。玉米、马铃薯、番茄都是从美洲传入旧大陆的。注意区分"美洲→旧大陆"和"旧大陆→美洲"两个方向的物种交换。

7.导致美洲原住民人口大量减少的最主要原因是（　）3分

  1. 欧洲人的屠杀
  2. 传染病的传播
  3. 过度劳动
  4. 饥荒

答案：B

欧洲人带来的天花、麻疹、流感等传染病是导致美洲原住民人口锐减的最主要原因。美洲原住民与旧大陆隔绝了约1.5万年，对这些传染病毫无免疫力，感染后死亡率极高。据估计，美洲原住民人口从约5000万骤减至不足1000万。

8.18世纪中期以后，成为世界上最大殖民帝国的国家是（　）3分

  1. 西班牙
  2. 葡萄牙
  3. 荷兰
  4. 英国

答案：D

殖民霸权的更替顺序：16世纪→葡萄牙和西班牙，17世纪→荷兰，18世纪中期起→英国。英国通过击败西班牙无敌舰队（1588）、三次英荷战争、英法七年战争（1756—1763）等，最终确立了全球殖民霸权，成为"日不落帝国"。

9."三角贸易"中获利最大的是（　）3分

  1. 非洲商人
  2. 美洲种植园主
  3. 欧洲商人
  4. 美洲原住民

答案：C

"三角贸易"中，欧洲商人作为贸易的组织者和主导者获利最大。他们用廉价的工业品在非洲换取奴隶，将奴隶高价卖给美洲种植园主，再将美洲的原料和金银运回欧洲高价出售，每一程都能获得巨额利润。这些利润成为欧洲资本原始积累的重要来源。

10.新航路开辟对非洲产生的最深远影响是（　）3分

  1. 引进了新的农作物
  2. 基督教得到广泛传播
  3. 丧失了大量精壮劳动力和持续数百年的殖民掠夺
  4. 建立了现代国家制度

答案：C

长达约300年的奴隶贸易使非洲丧失了大量精壮劳动力（约1200万人被贩卖到美洲），严重破坏了非洲的社会结构和发展进程，是非洲近代落后的重要历史原因之一。

11.下列关于"价格革命"的表述，正确的是（　）3分

  1. 使欧洲商品价格下降
  2. 有利于封建地主阶级
  3. 加速了封建制度的衰落和资本主义的发展
  4. 主要发生在亚洲

答案：C

殖民者从美洲掠夺的大量金银流入欧洲，导致欧洲货币贬值、物价飞涨——这就是"价格革命"。固定收取地租的封建地主实际收入下降，而从事商品生产和贸易的资产阶级获利增加。因此价格革命加速了封建制度的衰落和资本主义的发展。A、B项说反了；D项不正确——价格革命主要发生在欧洲。

12.荷兰在17世纪被称为"海上马车夫"，主要原因是（　）3分

  1. 造船技术最先进
  2. 拥有最庞大的海军
  3. 垄断了全球海上贸易运输
  4. 拥有最广阔的殖民地

答案：C

17世纪荷兰拥有世界上最庞大的商船队（数量超过英法总和），垄断了全球大部分海上贸易运输，因此被称为"海上马车夫"。荷兰的优势在于商业而非军事——海军和殖民地面积都不如后来的英国。

13.新航路开辟后，世界历史发生的根本性变化是（　）3分

  1. 欧洲确立了资本主义制度
  2. 世界从分散走向整体，全球联系初步建立
  3. 工业革命在欧洲率先发生
  4. 亚非拉国家开始工业化

答案：B

新航路开辟最根本的变化是"世界从分散走向整体"——各大洲通过贸易、移民、殖民活动连为一体。A项资本主义制度确立要等到17—18世纪资产阶级革命；C项工业革命在18世纪后期；D项亚非拉国家工业化是19世纪以后的事。

14.英法七年战争（1756—1763）的结果表明（　）3分

  1. 法国成为世界最大殖民帝国
  2. 英国确立了全球殖民霸权
  3. 荷兰重新成为海上强国
  4. 西班牙恢复了海上霸权

答案：B

七年战争中英国获胜，通过《巴黎条约》夺取了法国在北美（加拿大）和印度的殖民地，英国由此成为世界上最大的殖民帝国。法国虽然保留了部分加勒比殖民地（如海地），但在全球范围上已无法与英国抗衡。

15.新航路开辟对世界市场的形成产生的影响是（　）3分

  1. 世界市场最终形成
  2. 世界市场的雏形开始出现
  3. 世界市场已经完全成熟
  4. 世界市场没有受到任何影响

答案：B

注意措辞的精确性：新航路开辟使"世界市场的雏形开始出现"（高中教材用语）。世界市场的最终形成要等到19世纪中后期（第二次工业革命完成后）。关键词："雏形"而非"最终形成"。

四、材料分析题（共50分）

16.（25分）阅读下列材料，回答问题。25分

材料一：1492年以后，旧大陆和新大陆之间发生了大规模的物种交换。美洲的玉米、马铃薯传入欧洲和中国后，极大地提高了粮食产量。旧大陆的马、牛、羊传入美洲，改变了原住民的生产方式。但与此同时，欧洲人带来的天花、麻疹等传染病在美洲肆虐，造成了毁灭性的后果。
材料二："三角贸易"持续了约300年。据估计，约1200万非洲人被贩卖到美洲，更多的人在猎奴过程中丧生。一位英国商人在日记中写道："一个奴隶的利润足以让我们全家过上好日子。"这种贸易为欧洲带来了巨额财富，成为资本原始积累的重要途径。

（1）根据材料一，概括"哥伦布大交换"对世界的两方面影响。（6分）

（2）根据材料二，分析"三角贸易"对非洲和欧洲分别产生了什么影响。（8分）

（3）综合两则材料，说明新航路开辟对不同地区的不同影响。（6分）

（4）从"哥伦布大交换"和"三角贸易"中，你获得哪些历史认识？（5分）

（1）两方面影响（6分）：正面影响：物种交流促进了全球农业发展和人口增长（如玉米、马铃薯提高了粮食产量；马、牛改善了美洲生产方式）。负面影响：传染病传入美洲造成原住民人口大量死亡。

（2）对非洲和欧洲的影响（8分）：对非洲：丧失了大量精壮劳动力（约1200万奴隶被贩卖），社会结构遭到严重破坏，长期制约非洲发展，是非洲近代落后的历史根源之一。对欧洲：从奴隶贸易中获得巨额利润，完成了资本原始积累，推动了资本主义发展和工业革命的到来。

（3）对不同地区的不同影响（6分）：新航路开辟的影响具有不平衡性：对欧洲主要是正面的（获得财富和市场）；对美洲原住民和非洲人主要是负面的（人口锐减、文明毁灭、资源被掠夺）；对亚洲则是被动的、渐进的卷入（贸易扩大但也面临殖民威胁）。

（4）历史认识（5分，开放性）：①新航路开辟推动了全球联系，是人类历史的重大转折；②这种联系对各地区的影响截然不同，体现了历史发展的不均衡性；③殖民扩张带来了巨大的灾难和痛苦（尤其对非洲和美洲原住民）；④人类文明的发展进程中，要正视历史的不公正和伤痛。

17.（25分）阅读下列材料，回答问题。25分

材料：16世纪以来，欧洲殖民者在全球范围内展开了激烈的竞争。下表反映了殖民霸权的更替过程：
时期霸主国家主要殖民方式
16世纪葡萄牙、西班牙暴力掠夺、建立据点、直接统治
17世纪荷兰组建商业公司，垄断贸易
18世纪中期起英国商业公司+殖民战争+直接统治

（1）根据材料，概括殖民霸权更替的基本趋势。（6分）

（2）葡萄牙和西班牙的殖民方式有何不同？分别举一例说明。（6分）

（3）英国能够成为最大殖民帝国的条件有哪些？（7分）

（4）殖民霸权更替对世界格局产生了什么影响？（6分）

（1）基本趋势（6分）：①殖民霸权从伊比利亚半岛（葡萄牙、西班牙）向西北欧（荷兰、英国）转移；②殖民方式从暴力掠夺向制度化、公司化的殖民经营转变；③殖民范围不断扩大，最终形成全球殖民体系。

（2）殖民方式不同（6分）：葡萄牙以"商站式"殖民为主，主要在亚非沿海建立据点控制贸易，如占领马六甲、租借澳门。西班牙以直接占领和统治为主，如在美洲建立总督制，直接统治辽阔的殖民地。

（3）英国的条件（7分）：①地理位置优越（岛国，有利于发展海上力量）；②击败竞争对手（无敌舰队、荷兰、法国）；③资本主义经济发达，有强大的经济基础；④建立东印度公司等商业组织；⑤海军力量强大。

（4）对世界格局的影响（6分）：①确立了以欧洲为中心的世界格局；②推动了世界市场的形成和发展；③加深了世界各地区之间的联系，但也加剧了不平等；④为工业革命后全球殖民体系的最终形成奠定了基础。
