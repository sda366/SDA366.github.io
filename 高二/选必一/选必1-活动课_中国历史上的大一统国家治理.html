中国历史上的大一统国家治理

全册综合活动课 · 总结与升华
一、活动课概要
★ 课程定位

本活动课是对选择性必修1《国家制度与社会治理》全册内容的综合与升华，围绕"大一统国家治理"这一核心主题展开。通过纵向梳理从秦汉到明清"大一统"治理体系的演进脉络，横向整合政治制度、官员选拔、法律教化、民族关系、经济制度、基层治理六大维度，引导学生构建整体性认知，深入理解中国历史上大一统国家治理的丰富经验与制度智慧。

核心概念
大一统 · 政治统一 · 民族交融
文化认同 · 制度创新
活动目标
综合运用全册所学，
分析大一统国家治理的经验与智慧
◆ 活动方式
  • 小组合作探究：以4-6人为一组，各小组围绕不同维度的探究任务展开研究
  • 史料研读与数据分析：运用教材史料、补充文献与图表进行实证分析
  • 汇报展示与交叉点评：各小组展示研究成果，跨组互评，促进多维理解
  • 总结反思：形成对本册知识体系的整体把握，撰写活动反思报告
二、全册知识整合：大一统治理的六大维度

从选择性必修1六大单元出发，梳理"大一统国家治理"在不同制度领域的体现与成就。

维度一

政治制度维度的"大一统"

从秦汉的三公九卿制到隋唐的三省六部制，再到明清的内阁与军机处，中央官制不断强化君主集权；地方行政从郡县制到行省制，中央对地方的控制日益严密。政治制度的大一统为国家统一治理提供了组织保障。

中央官制 郡县制 行省制 君主集权
维度二

官员选拔维度的"大一统"

从世卿世禄到察举制、九品中正制，再到科举制的创立与完善，选官制度日趋公平、开放。科举制通过统一考试标准，打破了门阀垄断，促进了社会流动，为大一统国家输送了大量治国人才，强化了文化认同。

科举制 察举制 社会流动 人才选拔
维度三

法律教化维度的"大一统"

从秦代"以法为教"到汉唐"礼法结合"，再到宋明"德主刑辅"，中国古代法律体系呈现出礼法融合、德教为先的特征。《唐律疏议》作为中华法系的代表，体现了大一统国家以法律维护社会秩序、以教化统一思想的治理逻辑。

礼法结合 唐律疏议 德主刑辅 中华法系
维度四

民族关系维度的"大一统"

从秦汉"和亲"与"征伐"并举，到唐代"羁縻府州"制度，再到元明清时期的民族政策与改土归流，历代王朝在"大一统"框架下探索多元民族治理模式。民族交融与多元一体的格局不断深化，为现代统一多民族国家奠定了基础。

羁縻府州 改土归流 民族交融 多元一体
维度五

经济制度维度的"大一统"

从秦汉统一货币、度量衡，到唐代两税法、明代一条鞭法、清代"摊丁入亩"，赋役制度不断改革以维系国家财政。货币从铜钱到纸币再到白银货币化，体现了大一统国家对经济生活的深度介入与调控能力。

赋役制度 货币统一 两条税法 一条鞭法
维度六

基层治理维度的"大一统"

从秦汉的乡里制度到唐宋的保甲制度，再到明清的里甲与保甲并行，大一统国家对基层社会的管控不断精细化。宗族、乡约等民间力量与官方行政相互配合，形成了"皇权不下县，县下皆自治"的双层治理格局。

乡里制度 保甲制度 宗族乡约 双层治理
⚙ 大一统国家治理的逻辑链条
中央集权政治制度 → 统一选官培养人才 → 法律教化整合社会 → 民族政策扩展疆域
→ 财政经济提供支撑 → 基层治理稳固根基 → 大一统治理体系
三、探究活动设计

以下探究任务要求学生综合运用全册所学知识，在合作探究中深化对"大一统国家治理"的理解。

1

大一统国家治理的制度骨架

——中央与地方关系的制度演进

问题情境

公元前221年，秦始皇统一六国后，废除分封制，全面推行郡县制，奠定了此后两千余年大一统国家的地方治理基础。从秦到清，中央与地方关系经历了怎样的制度变迁？这些变迁如何体现"大一统"理念的深化？

探究步骤

  1. 回顾教材：整理秦郡县制、汉郡国并行、唐节度使、宋路制、元行省制、明清省制的基本特征
  2. 绘制"中央—地方关系演变时间轴"，标注关键转折点（如推恩令、藩镇割据、行省设立等）
  3. 分析比较：不同朝代在处理央地关系上有哪些"收权"与"放权"的交替？中央集权趋势是否不可逆转？
  4. 撰写800字以内的分析短文，以"从郡县到行省：大一统央地治理的制度逻辑"为题
参考材料： 教材第1-3课相关内容；《史记·秦始皇本纪》（节选）；《元史·地理志》（节选）；严耕望《中国地方行政制度史》相关论述

评价标准

优秀
史料运用充分，时间线完整，分析有深度，能提出独立见解
良好
基本覆盖主要制度，时间线清晰，分析较为全面
合格
罗列基本史实，时间线大致正确，分析略显表面
待提高
史实不完备，时间线混乱，欠缺比较分析
2

以文教化与以法治国

——礼法结合传统下的社会整合机制

问题情境

《唐律疏议》开篇即言"德礼为政教之本，刑罚为政教之用"，点明了中国古代"礼法结合"的治理智慧。在科举制普及、儒学成为官方意识形态的背景下，法律与教化如何协同发挥作用，将大一统的价值观念渗透到社会各阶层？

探究步骤

  1. 研读教材第8-10课，梳理"礼法结合"传统从汉代"春秋决狱"到唐代《唐律疏议》再到明清"律例并行"的演变
  2. 分析科举制度与儒学传播的关系：科举如何强化了"大一统"的文化认同？
  3. 选取一个具体案例（如唐代"不孝"入律、宋代"乡约"的推广），说明法律与教化如何相互配合
  4. 小组研讨：以"法律统一与文化大一统"为主题，准备5分钟专题发言
参考材料： 《唐律疏议·名例律》（节选）；《宋刑统》相关条目；朱熹《朱子家礼》；瞿同祖《中国法律与中国社会》；教材第8-10课

评价标准

优秀
案例分析深入，善用一手史料，能阐释礼法融合的内在逻辑
良好
梳理清晰，案例与论述匹配，体现法律与教化互动关系
合格
基本史实正确，案例选取合理，分析较为平直
待提高
法律与教化两条线索未能有机结合，论述单薄
3

多元一体的民族治理智慧

——从羁縻到改土归流的治理实践

问题情境

中国作为一个统一的多民族国家，历代王朝在治理边疆民族问题上积累了丰富的经验。从唐代的羁縻府州到元代的宣政院制度，再到清代的改土归流与驻藏大臣，大一统国家如何在"因俗而治"与"强化统一"之间取得平衡？这些实践对当代民族治理有何启示？

探究步骤

  1. 研读教材第11-13课，梳理历代民族政策的主要类型（羁縻、和亲、军事征服、行政整合等）
  2. 以小组为单位，选择一个边疆地区（如西域、西藏、西南、蒙古）进行专题研究
  3. 从"治理目标—治理方式—治理效果"三个维度评估该地区在不同朝代的治理实践
  4. 制作一张"中国古代民族治理方式比较表"，并撰写300字左右的评析
参考材料： 教材第11-13课；《汉书·西域传》；《大唐西域记》（节选）；《清实录》中关于改土归流的记载；费孝通《中华民族多元一体格局》（导论部分）

评价标准

优秀
专题研究有深度，比较分析透彻，能提出具有当代启示的观点
良好
分类清晰，比较维度合理，评析有理有据
合格
基本掌握各时期民族政策，比较表完整
待提高
内容停留在史料罗列，缺乏比较与提炼
4

基层社会如何支撑大一统

——赋役、户籍与乡村治理的制度网络

问题情境

大一统国家的运转需要庞大的财政支撑和稳定的基层社会。从秦汉的编户齐民与"租庸调制"，到唐代两税法，再到明代一条鞭法与清代"摊丁入亩"，赋役制度不断改革；与此同时，乡里保甲、宗族乡约等基层制度将国家权力延伸到县以下。大一统国家的"毛细血管"是如何渗透到基层社会的？

探究步骤

  1. 研读教材第16-18课（经济制度）和第19-21课（基层治理），梳理赋役制度与基层制度的对应关系
  2. 制作一张"赋役改革与社会治理联动关系图"，标注关键改革及其对基层的影响
  3. 讨论：赋役改革（如两税法、一条鞭法）如何影响了基层治理结构？国家汲取能力与社会稳定之间存在怎样的张力？
  4. 以"国家与乡村：大一统治理的基层之维"为题，撰写600字左右的小论文
参考材料： 教材第16-21课；《文献通考·田赋考》（节选）；《明史·食货志》（节选）；萧公权《中国乡村》；秦晖《传统十论》相关篇章

评价标准

优秀
联动关系梳理清晰，论证有层次，能触及制度张力与深层逻辑
良好
制度梳理完整，能体现赋役与基层治理的互动关系
合格
基础知识准确，图表基本反映制度演变
待提高
两条线索割裂，缺乏制度间的关联分析
跨组交流与成果整合

各小组完成探究任务后，安排两课时进行成果展示与交叉点评。每组展示时间8-10分钟，其他小组可就史料运用、论证逻辑、结论合理性等方面提问或补充。全部展示结束后，全班共同绘制"大一统国家治理六维关系图"，将六个维度的探究成果整合为一幅完整的知识图谱。

四、课后参考：进一步阅读与思考

📚 推荐阅读（拓展深化）

  • 钱穆《中国历代政治得失》—— 以简明笔法梳理历代政治制度的得失，对理解大一统治理框架有重要启发
  • 严耕望《中国地方行政制度史》—— 系统研究秦汉至隋唐地方行政制度的经典之作
  • 瞿同祖《中国法律与中国社会》—— 从社会学视角分析传统法律与社会的互动关系
  • 费孝通《中华民族多元一体格局》—— 解析中华民族形成与发展的民族学经典
  • 黄仁宇《万历十五年》—— 以1587年为切面，观察明代大一统治理体系的运行实态
  • 萧公权《中国乡村》—— 系统研究明清时期乡村社会控制与治理的权威著作

🔍 延伸思考（可任选一题撰写拓展论文）

  • 题目一：比较秦汉与明清两个大一统时期在治理模式上的异同，并分析其历史演进逻辑。
  • 题目二：从科举制度看"大一统"与文化认同的关系——以唐宋变革为中心。
  • 题目三："因俗而治"与"强化统一"之间的张力——以西南地区改土归流为例。
  • 题目四：中国古代赋役制度改革与基层治理结构的变迁——以一条鞭法为中心。
  • 题目五：综合全册所学，谈谈你对"大一统"这一中国历史治理智慧的理解。

📐 研究工具与方法建议

  • 制度比较法：从结构、功能、演进三个维度对不同时期的制度进行比较分析
  • 长时段分析法：借鉴布罗代尔的"长时段"理论，观察制度的长期变迁趋势
  • 史料批判法：关注一手史料的来源、作者意图与时代背景，培养批判性思维
  • 数据可视化：运用时间轴、思维导图、统计图表等形式呈现制度演变脉络
