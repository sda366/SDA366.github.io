1. 洋务运动
背景：第二次鸦片战争后，面对内忧外患的形势，清政府一些比较开明的官员主张利用西方先进技术，强兵富国，维护清王朝统治。

洋务派代表人物：

  • 中央：恭亲王奕䜣
  • 地方：曾国藩、李鸿章、左宗棠、张之洞等人

时间：从19世纪60年代到90年代中期

口号："自强""求富"

主要措施
方面 时间 内容 代表企业/措施

（"自强"）		 19世纪60年代起 创办近代军事工业 江南机器制造总局、福州船政局、安庆内军械所
民用企业
（"求富"）		 19世纪70年代起 开办近代民用企业，辅助军事工业 轮船招商局、开平煤矿、汉阳铁厂、湖北织布局

军事改革 19世纪60年代起 组建新式洋枪队；采用西式兵操练兵 淘汰传统兵器，武器和战术逐渐变化

重要提示：洋务派办的军事工业全是官办性质，不能自主经营，管理方式存在官僚衙门的恶习。以江南机器制造总局为例，非生产费用（薪水、工食、修建、办公等）占全部经费的40%左右，最高时达55%。生产成本高，效率低下，管理水平远低于国外同类企业。官督商办的民用企业中同样存在这类问题。
结果：经过30多年建设，中国近代军事工业、民用工业、交通运输业等有了一定程度的发展，在客观上促进了中国民族资本主义的产生。但由于清朝政治的腐败和外国势力的挤压，洋务运动未强求富的目标没有实现。
2. 美、日侵略台湾

背景：19世纪中后期，中国出现边疆危机，东南海疆首先面临外国威胁。美国早有觊觎台湾的野心。

  • 19世纪60年代：美国军舰进犯台湾，遭到当地人民坚决抵抗。此后，美国转而支持日本侵略台湾。
  • 琉球问题：琉球自明初起保持与中国的藩属关系。1879年，日本强占吞并琉球，改为日本冲绳县，清政府表示抗议，拒绝承认。
  • 1874年日军侵台：日本以琉球船民在台湾被杀为借口，进攻台湾南部。台湾人民据险反击，清政府派沈葆桢率兵入台。日本难以取胜，在英、美"调停"下，从中国勒索白银50万两后撤兵。
3. 收复新疆

背景：

  • 1865年：中亚浩罕国军事头目阿古柏率兵入侵新疆，占领南疆大部分地区和北疆部分地区。
  • 英、俄支持：英、俄两国无视中国主权，支持阿古柏侵略。俄国还于1871年出兵占领新疆伊犁地区。
  • 新疆各族人民反抗：阿古柏和英、俄两国的侵略，激起新疆各族人民的反抗。
海防与塞防之争：
  • 李鸿章（直隶总督）：认为海防重而塞防轻，主张将塞防经费移办海防。
  • 左宗棠（陕甘总督）：认为"东则海防，西则塞防，二者并重"，力主收复新疆。
  • 结果：清政府采纳左宗棠等人的意见。

收复过程：

  • 1875年5月：清政府任命左宗棠为钦差大臣，督办新疆军务。
  • 1876年：左宗棠指挥清军刘锦棠部入疆，采取先北后南、缓进急战的策略。
  • 在新疆各族人民支持和拥护下：清军于1878年消灭阿古柏侵略势力，收复除伊犁地区以外的新疆领土。
  • 收回伊犁：清政府通过艰难的外交谈判，收回被俄国侵占的伊犁地区大部分领土，但割让霍尔果斯河以西1万多平方千米土地，并付出大量赔款。
新疆建省（1884年）：清政府根据左宗棠的建议，在新疆地区建立行省，取"故土新归"之意，改称西域为"新疆"。新疆建省加强了中央对新疆的管辖，巩固了西北边防，维护了边疆稳定和国家统一。
4. 中法战争

背景：19世纪中后期，法国侵略越南后，又将矛头指向中国。

  • 驻扎在中越边境的黑旗军首领刘永福，应越南邀请，率部助越抗法，多次取得重大胜利。清政府后来也应邀派军队入越抗法。

战争进程：

  • 1883年12月：法国对驻守越南北部的清军发动进攻，中法战争爆发。
  • 1884年8月（马尾海战）：法国舰队闯入福建闽江口马尾，对中国福建水师发动突然袭击。福建水师仓促应战，大部分军舰被击沉，马尾船厂遭到严重破坏。清政府被迫对法宣战。
  • 台湾战事：法军占领台湾基隆，进犯淡水。督办台湾军务大臣刘铭传率部反击，取得胜利。法国舰队对台湾进行海上封锁。法军还袭扰浙江镇海，被中国守军击退，孤拔被击伤毙命。
  • 1885年初（镇南关大捷）：法军攻陷越南北部的谅山，并进犯中越边境的镇南关。老将冯子材率守军英勇抵抗，取得镇南关大捷，并收复谅山等地，扭转了战局。
战争结局：正当中越两国抗法斗争即将取得胜利时，清政府派李鸿章和法国代表在天津签订中法《越南条款》，承认越南受法国"保护"，同意中越边界开埠通商，中国以后修筑铁路要同法国人"商办"等条件。从此，法国的势力侵入中国云南、广西。
台湾建省（1885年）：中法战争使清政府进一步认识到台湾战略地位的重要性。1885年，清政府决定建立台湾行省。台湾建省加强了台湾的海防地位，强化了对台湾的治理，推动了当地社会经济发展。
知识连贯性补充

1. 与第一单元（中国开始沦为半殖民地半封建社会）的联系：

  • 两次鸦片战争的失败，使清政府中的一些开明官员认识到西方的先进，这是洋务运动兴起的直接背景。
  • 边疆危机的加剧（美、日侵台，俄占伊犁，法侵越南）与两次鸦片战争后列强瓜分中国的野心一脉相承。

2. 单元主题联系——"早期现代化的初步探索"：

  • 洋务运动是中国近代化的开端，开启了工业、军事、教育、交通等领域的现代化尝试。
  • 但洋务运动只学技术不学制度，具有局限性，为后来的戊戌变法和辛亥革命埋下伏笔。

3. 单元主题联系——"民族危机加剧"：

  • 19世纪60-80年代，中国面临严重的边疆危机，美、日、俄、法等列强从四面八方加紧侵略。
  • 左宗棠收复新疆和冯子材镇南关大捷体现了中国人民反抗外来侵略的英勇斗争精神。

4. 与后续课程的联系：

  • 洋务运动的失败（只学技术不学制度）为甲午中日战争的惨败埋下伏笔。
  • 中法战争"不败而败"的结局，暴露了清政府的腐朽和妥协，刺激了列强进一步瓜分中国的野心。
二、重点难点评析
（一）深度评析
1. 洋务运动的本质与局限性

洋务运动的根本目的是维护清王朝的封建统治，而不是发展资本主义。洋务派主张"中体西用"，即只学习西方的先进技术（器物层面），不改变中国封建专制制度（制度层面）。这种"只变其器，不变其道"的改革思路，决定了洋务运动不可能从根本上改变中国的落后面貌。

洋务企业具有浓厚的封建性和垄断性：军事工业全是官办，管理腐败，效率低下；民用企业"官督商办"，官府控制大权，商人没有经营自主权。这种体制严重束缚了生产力的发展。

2. 洋务运动与民族资本主义的关系

洋务运动在客观上促进了中国民族资本主义的产生：引进了西方先进生产技术；培养了一批技术人才；刺激了民间投资近代工业的热情。

但洋务运动对民族资本主义也有压制作用：洋务企业享有政府特权和垄断地位，挤压了民族资本主义的发展空间。

3. 海防与塞防之争的实质

表面上看是国防重点的争论，实质反映了清政府在国力衰微背景下的战略困境。李鸿章主张放弃新疆、专重海防，代表了妥协派的声音；左宗棠主张"二者并重"，体现了维护国家领土完整的坚定立场。清政府的最终决策表明，在国防战略上仍有一定的民族意识。

4. 中法战争"不败而败"的深刻教训

中国在军事上取得镇南关大捷，战场形势有利，但清政府却主动求和，签订了丧权辱国的《越南条款》。这充分暴露了清政府的腐朽无能和妥协投降的本质，也说明在封建专制制度下，即使有局部的军事胜利，也无法改变整体被动挨打的局面。

（三）高度评析
1. 从近代化道路的角度审视洋务运动

洋务运动是中国近代化的第一次尝试，但它走的是一条"器物层面"的近代化道路。历史证明，在不改变封建专制制度的前提下，单纯引进西方技术是不可能实现国家富强的。这一教训为后来的戊戌变法（制度层面的改良）和辛亥革命（制度层面的革命）提供了历史依据。

2. 从国家主权与领土完整的高度认识边疆危机

19世纪中后期，中国面临全面的边疆危机，列强从东南西北各个方向加紧侵略。左宗棠收复新疆和在台湾、镇南关的抵抗，体现了维护国家主权和领土完整的正义性。新疆建省（1884年）和台湾建省（1885年）是清政府加强边疆管理、维护国家统一的重要举措，具有深远的历史意义。

3. 从世界历史发展大势看中国的被动处境

19世纪中后期，西方列强进入帝国主义阶段，加紧了对亚非拉地区的殖民扩张。中国作为落后的封建大国，成为列强争夺的对象。洋务运动试图通过"师夷长技"来自救，但在资本主义世界体系已经形成的历史背景下，一个落后的封建国家很难通过局部改良来摆脱被侵略的命运。

4. 李鸿章"纸糊的老虎"自评的历史启示

李鸿章晚年评价自己的洋务事业是"纸糊的老虎"，这一评价深刻揭示了洋务运动的本质缺陷：没有坚实的制度基础，仅靠引进一些西方技术，就像用纸糊的老虎一样，外表吓人，实则不堪一击。这也反映了洋务派代表人物晚年对自身事业局限性的清醒认识。

三、教材栏目解答
想一想：新疆和台湾建省具有怎样的意义？

【参考答案】

一、新疆建省的意义（1884年）：

  1. 加强中央管辖：新疆建省加强了中央对新疆的管辖，改变了以往新疆地区由地方伯克（贵族）管理的松散状态。
  2. 巩固西北边防：有效抵御了沙俄等外来势力的侵略，维护了西北边疆的安全。
  3. 维护国家统一：巩固了西北边防，维护了边疆稳定和国家统一。
  4. 促进当地发展：推行与内地一致的行政制度，促进了新疆地区的经济开发和社会进步。

二、台湾建省的意义（1885年）：

  1. 加强海防地位：中法战争使清政府认识到台湾战略地位的重要性，建省加强了台湾的海防地位。
  2. 强化行政管理：改变台湾长期隶属于福建省的行政格局，提升为与内地同等地位的省份。
  3. 推动经济发展：强化了对台湾的治理，推动了当地社会经济发展。
  4. 具有前瞻性：体现了清政府对海防重要性的认识，是对日本等列强侵略野心的回应。

三、总体认识：

新疆和台湾建省是清政府在边疆危机日益严重的背景下，加强边疆管理、维护国家统一的重要举措。虽然清政府整体上腐朽无能，但在具体边疆问题上仍有一定的民族意识和国家观念。这两次建省对维护中国领土完整具有深远的历史意义，也为今天中国坚持国家统一和领土完整提供了历史依据。

材料研读
材料
李鸿章晚年这样评价自己的洋务事业：

"我办了一辈子的事，练兵也，海军也，都是纸糊的老虎，何尝能实在放手办理？不过勉强涂饰，虚有其表……"

——吴永《庚子西狩丛谈》
问题：这段话反映了李鸿章什么观点？你怎样看待他的这种说法？

【参考答案】

一、反映的观点：

  1. 自我否定：李鸿章认为自己的洋务事业（练兵、建海军）只是"纸糊的老虎"，外表吓人但内里空虚。
  2. 认识局限：他认识到洋务运动无法"实在放手办理"，受到各种制约，只能"勉强涂饰，虚有其表"。
  3. 无奈与悲观：反映了李鸿章晚年对洋务运动失败结局的无奈和悲观情绪。

二、对这种说法的认识：

（1）合理之处：

  • 洋务运动确实没有触及封建专制制度这个根本问题，只是在器物层面学习西方，如同"纸糊的老虎"，不可能真正实现国家富强。
  • 洋务派受到封建顽固派的阻挠、清政府的腐朽、外国势力的挤压等多重制约，确实无法"实在放手办理"。
  • 甲午战争中北洋舰队全军覆没，证明了李鸿章"纸糊的老虎"之说有一定道理。

（2）局限性：

  • 李鸿章将失败主要归因于外部制约，没有深刻认识到洋务运动自身"中体西用"指导思想的根本性缺陷。
  • 作为洋务运动的主要领导者，李鸿章对运动中的腐败、管理混乱等问题也负有不可推卸的责任。
  • 这种说法带有一定的推卸责任成分。

（3）总体评价：

李鸿章的说法是他晚年对洋务运动失败的沉痛反思，具有一定的历史认识价值。但他没有认识到，在不改变封建制度的前提下，任何技术层面的改良都不可能成功。这一历史任务最终由后来的资产阶级革命派来完成。

课后活动

1. 洋务运动为什么没有实现求强求富的目标？

【参考答案】

根本原因：洋务运动的根本目的是维护清王朝的封建统治，而不是改变封建制度。

具体原因：

  1. 指导思想错误：坚持"中体西用"，只学技术不学制度。在不改变封建专制制度的前提下，单纯引进西方技术不可能实现国家富强。
  2. 企业内部腐败：洋务企业封建衙门式管理，冗员充斥，贪污腐败严重。非生产费用占经费的40%-55%，成本高、效率低。
  3. 缺乏自主权：军事工业全是官办，不能自主经营；民用企业"官督商办"，商人无经营自主权，束缚了生产力发展。
  4. 外国势力挤压：列强不希望中国真正强大，通过各种方式制约中国近代工业的发展。
  5. 顽固派阻挠：封建顽固派反对任何改革，洋务派只能在有限范围内推行改革。
  6. 缺乏统筹规划：洋务运动由地方督抚各自为政，缺乏中央统一规划和协调，造成资源浪费和重复建设。

知识拓展：京师同文馆

【内容梳理】

  • 地位：京师同文馆是中国第一所官办的新式学堂。
  • 背景：奕䜣考虑到中外"语言不通，文字难辨"，急需培养外语人才，奏请在京师设立同文馆。
  • 成立时间：1862年正式成立。
  • 招生：选拔14岁以下的八旗子弟进馆学习，后又扩招满汉学员。
  • 课程：初设英、法、俄文，后增添算学、天文等。
  • 作用：对培养翻译人才和传播西学起了积极作用。

【拓展认识】

京师同文馆的设立标志着中国教育近代化的开端，是洋务运动在教育领域的重要举措。但它主要培养翻译和实用技术人才，而非具有独立思想的新式知识分子，这也是洋务运动"中体西用"指导思想在教育领域的体现。

四、课堂训练
（一）判断题 中等难度
1. 洋务运动的根本目的是发展资本主义，使中国走上富强的道路。

答案：错误（×）

解析：洋务运动的根本目的是维护清王朝的封建统治，而不是发展资本主义。洋务派是地主阶级的代表，他们学习西方技术的目的是"强兵富国"，维护清朝统治。虽然洋务运动在客观上促进了中国民族资本主义的产生，但这不是洋务派的初衷。

2. 洋务派在中央以奕䜣为代表，在地方以曾国藩、李鸿章、左宗棠、张之洞等人为代表。

答案：正确（√）

解析：符合教材原文表述。恭亲王奕䜣是洋务派在中央的代表人物，曾国藩、李鸿章、左宗棠、张之洞是地方上的代表人物，被称为"晚清四大名臣"。

3. 1885年，清政府在新疆地区建立行省；1884年，清政府决定建立台湾行省。

答案：错误（×）

解析：时间颠倒。1884年在新疆地区建立行省，1885年决定建立台湾行省。需要注意区分两个建省的时间。

4. 中法战争中，中国军队在镇南关取得大捷，但清政府却主动求和，签订了丧权辱国的《越南条款》。

答案：正确（√）

解析：1885年初，冯子材取得镇南关大捷，收复谅山等地，扭转了战局。但清政府却在有利形势下派李鸿章与法国签订《越南条款》，承认越南受法国"保护"。这就是历史上著名的"不败而败"。

5. 洋务运动期间创办的京师同文馆是中国第一所官办的新式学堂，主要培养翻译和军事人才。

答案：错误（×）

解析：京师同文馆主要培养翻译人才，而不是军事人才。军事人才主要由福州船政学堂等军事学校培养。同文馆的课程初设英、法、俄文，后增添算学、天文等，主要侧重于外语和科技翻译。

（二）选择题 较高难度
1. 19世纪60年代，某国军舰进犯台湾，遭到当地人民坚决抵抗。此后，该国转而支持日本侵略台湾。这个国家是（ ）

答案：B. 美国

解析：根据教材内容，19世纪60年代，美国军舰进犯台湾，遭到当地人民坚决抵抗。此后，美国转而支持日本侵略台湾。美国早有觊觎台湾的野心，自己直接侵略受阻后，便利用日本作为工具来达到侵略目的。

2. 洋务派创办近代军事工业的口号是（ ）

答案：B. "自强""求富"

解析：洋务派从19世纪60年代起以"自强"为口号创办近代军事工业，从19世纪70年代起又以"求富"为口号开办近代民用企业。"民主""科学"是新文化运动的口号；"扶清灭洋"是义和团的口号；"变法图强"是维新派的口号。

3. 下列关于洋务运动的表述，正确的是（ ）

答案：C. 洋务运动在客观上促进了中国民族资本主义的产生

解析：A错误，洋务运动未强求富的目标没有实现；B错误，洋务运动坚持"中体西用"，没有改变封建专制制度；D错误，洋务运动受到封建顽固派的强烈反对。C正确，洋务运动引进了西方先进技术，培养了一批技术人才，在客观上促进了中国民族资本主义的产生。

4. 1884年，清政府在新疆地区建立行省。这一举措的主要目的是（ ）

答案：B. 加强中央对新疆的管辖，巩固西北边防

解析：根据教材，新疆建省加强了中央对新疆的管辖，巩固了西北边防，维护了边疆稳定和国家统一。虽然建省也促进了当地发展（A），但主要目的是加强管辖和巩固边防。C与建省目的无关；D时间错误，甲午战争发生于1894年。

5. 中法战争期间，清军取得镇南关大捷的将领是（ ）

答案：C. 冯子材

解析：1885年初，老将冯子材率守军在镇南关英勇抵抗，取得镇南关大捷，并收复谅山等地，扭转了战局。左宗棠是收复新疆的将领；李鸿章是签订《越南条款》的代表；刘铭传是在台湾抗击法军的将领。

6. 下列关于洋务运动失败原因的表述，不准确的是（ ）

答案：C. 洋务运动得到了西方列强的大力支持

解析：本题要求选择"不准确"的表述。A、B、D都是洋务运动失败的正确原因。C表述错误——西方列强并不希望中国真正强大，它们通过各种方式制约中国近代工业的发展，如控制技术、挤压市场等。列强对洋务运动的态度是既想从中国获利，又不愿中国富强，所以不可能"大力支持"。
