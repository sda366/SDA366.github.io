八年级历史上册 · 第七单元 解放战争

第20课 国民党挑起内战

2024修订版 · 人教版/统编版

一、学习目标

  • 了解重庆谈判的背景与《双十协定》
  • 理解全面内战爆发的标志和过程
  • 掌握解放区土地改革的内容与意义
  • 认识民心向背是战争胜负的决定因素

二、知识点时间线

1945.8 毛泽东在周恩来、王若飞等陪同下赴重庆谈判，经过43天谈判，国共双方签署《双十协定》（即《政府与中共代表会谈纪要》）。
1946.6 蒋介石撕毁政协决议，国民党军队大举进攻中原解放区，全面内战爆发。
1947.3 国民党军改变策略，发动对陕北解放区和山东解放区的重点进攻。
1947.3 毛泽东、周恩来等主动撤离延安，转战陕北，继续指挥全国解放战争。
1947.3-5 彭德怀指挥西北野战军先后取得青化砭、羊马河、蟠龙等战役胜利，粉碎国民党对陕北的重点进攻。
1947.夏 刘伯承、邓小平率晋冀鲁豫解放军主力挺进大别山，揭开人民解放军战略进攻的序幕。

三、重难点评析

【深度】重庆谈判的背景和历史意义

背景：抗日战争胜利后，全国人民渴望和平、民主。但以蒋介石为首的国民党，在美国的支持下，阴谋发动反共反人民的内战。由于内战准备不充分，同时也为了掩盖其真面目，蒋介石三次电邀毛泽东到重庆商谈国内和平问题。

中共的态度：中共中央以斗争求和平，决定派毛泽东赴重庆谈判，以争取和平民主，揭露蒋介石"假和平、真内战"的阴谋。

结果：1945年10月10日签署《双十协定》——坚决避免内战，建设独立、自由、富强的新中国。但国民党始终不承认解放区民主政权和人民军队的合法地位。

意义：重庆谈判和《双十协定》的签订，表明中共争取和平民主的诚意，在全国人民面前揭露了蒋介石"假和平、真内战"的本质，赢得了政治上的主动。

【广度】解放区土地改革

1947年，中国共产党召开全国土地会议，颁布《中国土地法大纲》，规定：没收地主土地，废除封建剥削的土地制度，实行耕者有其田，按照农村人口平均分配土地。

总路线：依靠贫雇农，团结中农，有步骤、有分别地消灭封建剥削制度，发展农业生产。

意义：土地改革使解放区广大农民获得了土地，激发了农民革命和生产的积极性，翻身农民踊跃参军参战，为解放战争的胜利提供了重要的人力、物力保障。

【高度】民心向背是战争胜负的决定因素

中国共产党在解放区实行土地改革，赢得了广大农民的拥护和支持——这是解放战争迅速取得胜利的根本原因。

国民党统治区物价飞涨、贪污腐败、苛捐杂税，民不聊生；而解放区政治清明、土地改革、人民拥护。两相对比，得民心者得天下，这是历史的必然规律。

四、教材栏目

想一想

毛泽东为什么要冒险赴重庆谈判？

[ 点击显示答案 ]

为了尽一切可能争取国内和平，揭露蒋介石假和平真内战的阴谋，团结和教育广大人民，赢得政治上的主动权。

材料研读

"……若不速予剿除，不仅抗战胜利之功，等于虚掷，且必遗害无穷，使中华民族永无复兴之可能，殊非国家民族之福。"

——蒋介石《剿匪手本》（1945年）

这段材料说明了什么？

[ 点击显示解析 ]

说明蒋介石在抗战胜利后仍然坚持反共立场，把中国共产党和人民军队视为"匪"，企图以武力消灭中共力量，其"和平"姿态只是欺骗舆论的手段。

课后活动

1. 查阅资料，了解重庆谈判中双方争论的焦点是什么。

2. 想一想：为什么说"挺进大别山"是人民解放军战略进攻的序幕？

[ 点击显示参考思路 ]

第1题：重庆谈判中双方争论的焦点是解放区政权和人民军队的合法地位问题。中共提出应承认解放区民选政府和人民军队的合法地位，但国民党方面拒绝承认。

第2题：挺进大别山标志着人民解放军由战略防御转向战略进攻。刘邓大军千里跃进大别山，直接威胁国民党统治的核心地区——南京和武汉，迫使国民党军从进攻转入防守，战争形势发生根本性转折。

知识拓展

"第二条战线"：国民党发动全面内战后，在国统区，中共领导了以学生运动为先导的人民民主运动。1946年冬北平学生抗议美军暴行运动、1947年上海工人反饥饿反内战运动等，形成了反对国民党统治的"第二条战线"，使国民党陷入全民包围之中。

答案：×

解析：《双十协定》虽然提出了和平建国的基本方针，但国民党始终没有承认解放区民主政权和人民军队的合法地位，国共矛盾并未彻底解决。

判断2. 全面内战爆发的标志是1946年6月国民党军队进攻中原解放区。
[ 点击显示答案 ]

答案：√

解析：1946年6月，蒋介石撕毁政协决议，国民党军队大举进攻中原解放区，全面内战由此爆发。

判断3. 1947年，中共中央主动撤离延安，标志着中共中央放弃了陕北根据地。
[ 点击显示答案 ]

答案：×

解析：撤离延安是中共中央为诱敌深入、在运动中寻机歼敌而采取的战略主动转移。毛泽东、周恩来等转战陕北，继续指挥全国解放战争，并未放弃陕北根据地。

二、选择题
选择1. 毛泽东赴重庆谈判的目的是（　）
  1. A. 满足蒋介石的要求，放弃武装斗争
  2. B. 尽一切可能争取和平，揭露蒋介石假和平的阴谋
  3. C. 借谈判拖延时间，为内战做准备
  4. D. 迫使蒋介石承认中共的合法地位
[ 点击显示答案 ]

答案：B

解析：毛泽东赴重庆谈判的目的是为了尽一切可能争取国内和平，同时揭露蒋介石假和平真内战的阴谋，团结和教育人民。

选择2. 1947年，揭开人民解放军战略进攻序幕的事件是（　）
  1. A. 中共中央主动撤离延安
  2. B. 青化砭战役
  3. C. 刘邓大军挺进大别山
  4. D. 粉碎国民党对山东解放区的重点进攻
[ 点击显示答案 ]

答案：C

解析：1947年夏，刘伯承、邓小平率晋冀鲁豫解放军主力千里挺进大别山，直接威胁国民党统治的核心地区，标志着人民解放军由战略防御转为战略进攻。

选择3. 1947年，中国共产党颁布的《中国土地法大纲》规定（　）
  1. A. 实行减租减息政策
  2. B. 没收地主土地，实行耕者有其田
  3. C. 实行土地私有化
  4. D. 将土地全部收归国有

答案：B

解析：1947年《中国土地法大纲》规定：没收地主土地，废除封建剥削的土地制度，实行耕者有其田，按农村人口平均分配土地。

选择4. 解放战争迅速取得胜利的根本原因是（　）
  1. A. 解放军的英勇善战
  2. B. 国民党军队士气低落
  3. C. 中国共产党赢得了广大人民的支持
  4. D. 美国放弃对国民党的援助
[ 点击显示答案 ]

答案：C

解析：中国共产党通过土地改革赢得了广大农民的拥护和支持，翻身农民踊跃参军参战，为解放战争提供了源源不断的人力、物力支持。民心向背是战争胜负的决定性因素。

三、材料分析题
材料

材料一："本人此次来渝，系应国民政府主席蒋介石之邀请，商讨团结建国大计……目前最迫切者，为保证国内和平，实施民主政治，稳定国内局势……"

——毛泽东在重庆机场的谈话（1945年8月28日）

 

材料二："……谈判结果，应承认蒋介石的抗日功绩，承认国民党在全国的领导地位……但同时要坚持斗争，对蒋介石的阴谋必须揭露……"

——中共中央对重庆谈判的指示精神

（1）材料一中毛泽东赴渝"系应"谁的邀请？此次谈判历史上称为什么？

（2）根据材料二，指出中共对蒋介石采取的策略是什么？

（3）结合所学知识，分析重庆谈判的历史意义。

[ 点击显示参考答案 ]

（1）应蒋介石的邀请。此次谈判被称为"重庆谈判"。

（2）中共对蒋介石采取"以斗争求团结"的策略——一方面承认其抗战功绩和领导地位，以争取和平；另一方面坚持原则，对其"假和平、真内战"的阴谋进行揭露和斗争。

（3）历史意义：
① 表明了中国共产党争取和平民主的诚意，赢得了全国人民的同情和支持；
② 签订了《双十协定》，使和平建国有了协议依据；
③ 揭露了蒋介石假和平真内战的本质，使其在政治上陷入被动；
④ 为中国共产党赢得了政治上的主动权和准备自卫战争的时间。

六、本课小结

抗日战争胜利后，国民党在美国支持下阴谋发动内战。中共为争取和平民主，毛泽东赴重庆谈判并签订《双十协定》，但在政治上揭露了蒋介石的真实面目。1946年全面内战爆发后，中共采取正确的战略战术，主动撤离延安、转战陕北，粉碎国民党重点进攻。1947年，刘邓大军挺进大别山，揭开战略进攻序幕。与此同时，解放区推行土地改革，赢得民心，为解放战争的最终胜利奠定了坚实基础。
