第3课 太平天国运动

八年级历史上册 · 第一单元 中国开始沦为半殖民地半封建社会

一、知识点总结（含时间线）
【太平天国运动大事年表】
1843年

创立拜上帝会：洪秀全创立拜上帝会，和冯云山前往广西传教

1851年1月11日

金田起义：洪秀全在广西桂平金田村发动武装起义，建号太平天国，起义军称太平军

1851年秋

永安建制：攻克永安后，洪秀全封王，太平天国初步建立起政权组织

1852年春

突围北上：从永安突围，围困桂林、长沙，攻克武昌，队伍发展至50万人

1853年3月

定都天京：攻占南京，改名天京，作为都城

1853年

颁布《天朝田亩制度》：颁布纲领性文件，并开始北伐和西征

1856年

军事全盛：控制长江沿岸城镇，掌握安徽、江西、湖北东部和江苏部分地区

1856年秋

天京事变：领导集团内讧，杨秀清被杀，韦昌辉被处死，石达开出走，太平天国由盛转衰

1859年

《资政新篇》：洪仁玕写成《资政新篇》，提出向西方学习、改革内政等主张

1860年

安庆陷落：曾国藩湘军包围安庆，陈玉成、李秀成救援失败，安庆陷落

1862年

天京被围：湘军围困天京，太平军激战40多天未能解围

1864年夏

天京陷落：洪秀全病逝，湘军攻入天京，太平天国运动失败

【洪秀全与金田起义】
起义背景
  • 外因：鸦片战争的失败，进一步加深了清政府的统治危机
  • 内因：随着剥削的加重，统治阶级与劳动群众之间的矛盾日益尖锐，农民的反抗斗争不断在各地发生
  • 组织基础：1847年10月，天地会在湘桂边界发动武装起义，声势浩大，为更大规模的革命风暴奠定了基础
洪秀全创立拜上帝会
  • 人物简介：洪秀全（1814—1864），广东花县人，农民家庭出身，做过乡村教师
  • 科举失败：多次到广州参加科举考试，都没有考中
  • 思想转变：在广州得到一部布道小册子《劝世良言》，在这部书启示下，创立拜上帝会
  • 核心主张：自称上帝次子，受命下凡来铲除妖魔，拯救百姓
  • 组织发展：劝说同学冯云山和族弟洪仁玕率先加入，随后杨秀清、萧朝贵、石达开、韦昌辉等骨干相继加入
金田起义
  • 时间：1851年1月11日
  • 地点：广西桂平金田村
  • 标志：洪秀全发动武装起义，建号太平天国，起义军称太平军
  • 领导核心：洪秀全称天王，封杨秀清为东王，萧朝贵为西王，冯云山为南王，韦昌辉为北王，石达开为翼王
  • 制度规定：东王以下诸王均受东王节制
  • 意义：太平天国初步建立起政权组织
【定都天京与军事全盛】
进军过程
  1. 1852年春：太平军从永安突围北上，先后围困桂林、长沙，攻克武昌
  2. 队伍壮大：虽然冯云山、萧朝贵在进军途中牺牲，但队伍发展至50万人
  3. 1853年春：太平军放弃武昌，分水陆两路，沿长江东下
  4. 1853年3月：攻占南京，将南京改名为天京，作为都城
进军迅速的原因
  • 群众基础好：沿途贫苦农民纷纷加入，队伍迅速壮大
  • 组织建设得力：组建土营（挖煤工）和水营（船民），为攻城和水上作战创造有利条件
  • 清军腐朽：清朝军队战斗力低下，无力阻挡
  • 民心所向：清政府剥削压迫严重，农民渴望改变现状
《天朝田亩制度》（1853年）

核心内容：

"凡分田照人口，不论男妇。算其家口多寡，人多则分多，人寡则分寡，杂以九等……凡天下田，天下人同耕，此处不足则迁彼处，彼处不足则迁此处。凡天下田，丰荒相通，此处荒，则移彼丰处以赈此荒处，彼处荒，则移此丰处以赈彼荒处……"

分配原则：规定不分男女，按人口和年龄平均分配土地

理想社会：建立"有田同耕，有饭同食，有衣同穿，有钱同使，无处不均匀，无人不饱暖"的理想社会

评价：

  • 积极方面：反映了千百年来农民要求得到土地的强烈愿望，对于发动和鼓舞广大农民起来参加反封建斗争起了积极作用
  • 局限性：带有浓厚的空想色彩，在长期的战争环境下未能推行
北伐和西征
军事行动 时间 目的 结果
北伐 1853年 推翻清朝统治 曾逼近天津，最后全军覆没
西征 1853年 巩固政权 取得重大胜利

军事全盛：到1856年，太平天国控制了从湖北武昌到江苏镇江的长江沿岸城镇，掌握了安徽、江西、湖北东部和江苏部分地区，军事上进入全盛时期。

【天京陷落与运动失败】
天京事变——由盛转衰
  • 原因：定都天京后，太平天国领导人开始享乐腐化，争权夺利
  • 时间：1856年秋
  • 经过：东王杨秀清逼洪秀全封他为"万岁"，意图篡位。结果引发天京事变
  • 结果：杨秀清被杀，韦昌辉被处死，石达开率部出走
  • 影响：太平军损失惨重，人心士气受到严重影响，太平天国由盛转衰
挽救措施
  • 提拔新将领：洪秀全封洪仁玕为干王，总理朝政；提拔陈玉成、李秀成等一批青年将领
  • 《资政新篇》：洪仁玕写成《资政新篇》，提出向西方学习、改革内政等一系列主张，得到洪秀全的赞赏
  • 未能实践：受当时历史条件限制，《资政新篇》未能付诸实践
  • 局部胜利：陈玉成、李秀成等协同作战，先后取得一系列胜利，但未能从根本上改变军事上的不利局面
《资政新篇》

作者：洪仁玕

核心主张：向西方学习、改革内政

评价：是先进的中国人最早提出的在中国发展资本主义的方案，具有鲜明的资本主义色彩。但由于当时中国缺乏必要的社会条件（没有坚实的经济基础、战争环境、缺乏群众基础），未能付诸实践。

中外反动势力联合镇压
  • 湘军：曾国藩编练的湘军成为太平军最凶悍的敌人。1860年，湘军包围安庆，陈玉成和李秀成救援失败，安庆陷落，陈玉成被俘就义
  • 淮军与洋枪队：李秀成率太平军攻克杭州，进逼上海，遭到华尔洋枪队和李鸿章所部淮军的抵抗和反扑
  • 天京被围：1862年，湘军围困天京，太平军激战40多天，未能解除威胁
  • 天京陷落：1864年夏，洪秀全病逝，湘军轰塌城墙冲入城内。守城将士大部分战死，其余突围。天京陷落，标志着太平天国运动失败
【太平天国运动的历史评价】
历史意义
  • 太平天国运动是中国历史上一次规模宏大的农民战争
  • 坚持斗争14年，转战大半个中国
  • 沉重打击了清朝的统治和外国侵略势力
  • 谱写了中国近代史上的壮烈篇章
失败原因
  • 阶级局限性：农民阶级不是新的生产力和生产关系的代表，无法克服小生产者所固有的阶级局限性
  • 纲领不切实际：无法提出切合实际的革命纲领
  • 领导集团腐败：无法制止和克服领导集团的腐败
  • 内部不团结：无法长期保持领导集团的团结（天京事变）
  • 中外联合镇压：在中外反动势力联合镇压下最终失败
知识连贯性补充

与第1、2课联系：太平天国运动是在鸦片战争后民族矛盾、阶级矛盾激化的背景下爆发的，与两次鸦片战争有直接的因果关系。列强侵略和清政府腐朽是运动爆发的重要背景。

与单元主题联系：第一单元"中国开始沦为半殖民地半封建社会"，太平天国运动既是这一社会背景下的产物，又反过来沉重打击了清政府和外国侵略势力，体现了中国人民的反抗精神。

与后续内容联系：太平天国运动失败后，清政府内忧外患，部分官员认识到必须学习西方才能挽救统治，由此引发了洋务运动。从太平天国到洋务运动，反映了近代中国探索救亡图存道路的不同尝试。

横向联系：曾国藩幕僚赵烈文预感清朝将亡。1867年赵烈文预言"不出五十年"清朝将土崩瓦解，40多年后1911年武昌起义果然推翻了清朝统治，印证了太平天国运动对清朝统治的沉重打击。

二、重点难点评析
第3课 太平天国运动
【深度评析】太平天国运动的性质与特点

1. 性质：旧式农民战争的最高峰

太平天国运动是中国历史上规模最宏大的一次农民战争，它不同于以往的农民起义：有组织（拜上帝会）、有纲领（《天朝田亩制度》）、有政权（太平天国）、有军队（太平军），持续时间最长（14年），波及范围最广（大半个中国）。

2. 反封建与反侵略的双重性质

太平天国运动不仅反对清朝封建统治，还抗击外国侵略势力（如抗击华尔洋枪队）。这是以往农民战争所不具备的新特点，体现了近代中国半殖民地半封建社会背景下的特殊性质。

3. 利用宗教发动群众的特点

洪秀全利用西方基督教改造而来的"拜上帝会"发动群众，这在农民战争史上是独特的。宗教既起到了组织和动员作用，也带有迷信色彩，成为后来领导集团分裂的思想因素之一。

4. 天京事变的深层原因

天京事变表面上是领导集团争权夺利的内讧，实质上是农民阶级局限性的集中体现。农民阶级不是先进生产力的代表，无法建立科学的领导体制和权力制衡机制，在取得一定胜利后容易腐化堕落。

【广度评析】《天朝田亩制度》与《资政新篇》的比较
比较项 《天朝田亩制度》（1853年） 《资政新篇》（1859年）
背景 定都天京后，为巩固政权、动员农民 天京事变后，为挽救危局
核心内容 按人口平均分配土地，建立理想农业社会 向西方学习，发展资本主义工商业
经济主张 小农经济基础上的绝对平均主义 发展近代工业、商业、交通运输业
性质 农民阶级的革命纲领，带有浓厚空想色彩 具有鲜明资本主义色彩的改革方案
群众基础 反映农民要求得到土地的强烈愿望，有广泛群众基础 未涉及农民最关心的土地问题，缺乏群众基础
实践结果 战争环境下未能推行 缺乏社会条件，未能付诸实践
历史地位 几千年来农民反封建斗争的思想结晶 先进的中国人最早提出的发展资本主义方案

共同点：都具有反封建的进步性；都因脱离实际或缺乏条件而未能实现。

【高度评析】从太平天国运动看近代中国农民阶级的历史命运

1. 农民阶级是反封建的主力军

太平天国运动证明，在半殖民地半封建的中国，农民阶级具有强烈的反封建革命性，是民主革命的主力军。他们承受了封建地主阶级和帝国主义的双重压迫，具有改变现状的强烈愿望。

2. 农民阶级无法领导中国革命走向胜利

农民阶级不是新的生产力和生产关系的代表，无法克服自身的局限性：提不出科学的革命纲领、无法制止领导集团腐化、无法保持长期团结、无法正确应对中外反动势力。这决定了单纯的农民战争不可能完成反帝反封建的历史任务。

3. 对后来革命的启示

太平天国运动的失败表明，中国革命需要先进阶级的领导，需要科学理论的指导。这为后来无产阶级（通过中国共产党）领导中国革命提供了历史依据。

4. "中外反动势力联合镇压"的深远影响

太平天国运动首次遭到中外反动势力的联合镇压，这一模式在此后的中国革命中反复出现（如义和团运动、辛亥革命等）。它揭示了在半殖民地中国，反动势力是如何勾结起来镇压人民革命的，也预示着中国革命必须同时反对帝国主义和封建主义的双重任务。

5. 历史遗产与精神价值

尽管太平天国运动最终失败，但它沉重打击了清朝统治和外国侵略势力，延缓了中国半殖民地化的进程。运动中所体现的反抗精神、平等理想、民族气节，成为中华民族宝贵的精神财富。

三、教材栏目解答
第3课 太平天国运动
【想一想】太平军进军为什么如此迅速？

答案：

  1. 群众基础广泛：鸦片战争后，清政府剥削加重，阶级矛盾尖锐，贫苦农民纷纷加入太平军，队伍迅速壮大至50万人
  2. 组织建设得力：组建土营（挖煤工）和水营（船民、纤夫），为攻城和水上作战创造了有利条件
  3. 清军腐朽无能：清朝军队战斗力低下，八旗、绿营兵不堪一战，无力阻挡太平军
  4. 战略得当：从永安突围北上，先后围困桂林、长沙，攻克武昌，然后沿长江东下，直取南京，战略路线清晰
  5. 民心所向：《天朝田亩制度》反映了农民要求得到土地的强烈愿望，得到广大农民拥护
【材料研读】
"凡分田照人口，不论男妇。算其家口多寡，人多则分多，人寡则分寡，杂以九等……凡天下田，天下人同耕，此处不足则迁彼处，彼处不足则迁此处。凡天下田，丰荒相通，此处荒，则移彼丰处以赈此荒处，彼处荒，则移此丰处以赈彼荒处……"

——《天朝田亩制度》

阅读材料，结合当时形势谈谈你对太平天国土地分配办法的看法。

答案：

一、积极方面

  1. 反封建的进步性：否定封建地主土地所有制，主张按人口平均分配土地，体现了农民阶级对封建剥削的反抗
  2. 调动革命积极性：反映了千百年来农民要求得到土地的强烈愿望，对于发动和鼓舞广大农民参加反封建斗争起了积极作用
  3. 平等思想："不论男妇"体现了一定的男女平等思想，"无处不均匀，无人不饱暖"体现了对理想社会的追求

二、局限性

  1. 空想色彩浓厚：在小农经济基础上实行绝对平均主义，违背了社会发展规律，是不可能实现的
  2. 违背经济规律：取消商品交换，实行"圣库制度"，否定了商品经济的发展
  3. 战争环境限制：在长期的战争环境下，根本没有条件实施土地分配
  4. 未能推行：事实上这一制度并未真正实行，太平天国的土地政策在实际操作中并没有严格执行

总结：《天朝田亩制度》是农民阶级的革命纲领，集中反映了农民反对封建剥削、要求获得土地的强烈愿望，具有鲜明的反封建意义。但由于其浓厚的空想色彩和时代局限，最终未能实现。

2. 《天朝田亩制度》是太平天国的纲领性文献。查阅相关资料，谈谈你的认识。

答案：

一、核心内容

《天朝田亩制度》规定不分男女，按人口和年龄平均分配土地，建立"有田同耕，有饭同食，有衣同穿，有钱同使，无处不均匀，无人不饱暖"的理想社会。

二、历史意义

  1. 是几千年来农民反封建斗争的思想结晶
  2. 否定了封建地主土地所有制，具有鲜明的反封建意义
  3. 反映了农民阶级对土地和美好生活的强烈向往
  4. 对于发动和鼓舞农民参加反封建斗争起了积极作用

三、局限性

  1. 绝对平均主义违背社会发展规律，具有浓厚的空想色彩
  2. 在小农经济基础上取消商品交换，违背了经济发展规律
  3. 战争环境下根本无法实施

四、总体评价

《天朝田亩制度》是农民阶级的革命纲领，既有反封建的进步性，又有脱离实际的空想性。它的提出和实施困境，集中反映了农民阶级作为小生产者的历史局限性。

【知识拓展】曾国藩幕僚赵烈文预感清朝将亡

答案：

1. 背景

太平天国给清朝统治以沉重打击。1867年7月的一天晚上，时任两江总督的曾国藩忧心忡忡地对其幕僚赵烈文谈论京城的形势。

2. 赵烈文的预言

赵烈文回答说："天下治安，一统久矣，势必驯至分剖。然主威素重，风采未开，若非抽心一烂，则土崩瓦解之局不成。以烈度之，异日之祸，必先根本颠仆，而后方州无主，人自为政，殆不出五十年矣。"

3. 预言的应验

40多年后，即1911年，伴随着武昌起义的枪炮声，清朝统治在各省独立的声浪中迅速土崩瓦解。赵烈文的预言惊人地应验了。

4. 历史启示

  • 太平天国运动虽然失败，但对清朝统治造成了致命打击
  • 清朝的灭亡是历史的必然，赵烈文的预言反映了有识之士对时局的清醒认识
  • 农民起义虽然无法单独完成革命任务，但为最终推翻封建统治创造了条件

三、材料题

材料一："凡分田照人口，不论男妇。算其家口多寡，人多则分多，人寡则分寡，杂以九等……凡天下田，天下人同耕……"

——《天朝田亩制度》
材料二："洪仁玕写成《资政新篇》，提出向西方学习、改革内政等一系列主张，得到洪秀全的赞赏。受当时历史条件限制，《资政新篇》未能付诸实践。"
材料三："太平天国运动是中国历史上一次规模宏大的农民战争。它坚持斗争14年，转战大半个中国，沉重打击了清朝的统治和外国侵略势力……由于农民阶级的局限性，太平天国无法提出切合实际的革命纲领，无法制止和克服领导集团的腐败，也无法长期保持领导集团的团结，在中外反动势力联合镇压下，最终失败。"
请回答：

（1）根据材料一，概括《天朝田亩制度》的核心内容，并分析其进步性和局限性。

（2）根据材料二，指出《资政新篇》的核心主张。为什么它"未能付诸实践"？

（3）根据材料三，概括太平天国运动的历史意义和失败原因。

（4）综合上述材料，结合所学知识，谈谈你对农民阶级在近代中国革命中作用的认识。

参考答案：

（1）核心内容：按人口平均分配土地，不分男女，建立"无处不均匀，无人不饱暖"的理想社会。
进步性：否定封建地主土地所有制，反映了农民要求得到土地的强烈愿望，具有鲜明的反封建意义。
局限性：带有浓厚的空想色彩，在小农经济基础上实行绝对平均主义，违背社会发展规律，在战争环境下未能推行。

（2）核心主张：向西方学习、改革内政，发展资本主义。
未能实践的原因：①当时中国缺乏发展资本主义的社会条件（没有坚实的经济基础）；②处于战争环境；③没有涉及农民最关心的土地问题，缺乏群众基础。

（3）历史意义：①是中国历史上规模宏大的农民战争；②坚持斗争14年，转战大半个中国；③沉重打击了清朝的统治和外国侵略势力。
失败原因：①农民阶级的局限性；②无法提出切合实际的革命纲领；③无法制止和克服领导集团的腐败；④无法长期保持领导集团的团结；⑤中外反动势力的联合镇压。

（4）认识：①农民阶级是反封建的主力军，具有强烈的革命性；②但农民阶级不是新的生产力和生产关系的代表，无法克服自身的局限性，不能领导中国革命取得胜利；③中国革命需要先进阶级的领导和科学理论的指导。

材料："定都天京后，太平天国领导人开始享乐腐化，争权夺利。1856年秋，东王杨秀清逼洪秀全封他为'万岁'，意图篡位。结果引发天京事变，杨秀清被杀，韦昌辉被处死，石达开率部出走，太平军损失惨重，人心士气受到严重影响。天京事变使太平天国由盛转衰。"
请回答：

（1）材料反映的是什么历史事件？发生在哪一年？

（2）根据材料，分析这一事件发生的原因。

（3）这一事件对太平天国运动产生了什么影响？

（4）结合所学知识，谈谈这一事件给我们的历史启示。

参考答案：

（1）天京事变。1856年秋。

（2）直接原因：太平天国领导人开始享乐腐化，争权夺利；东王杨秀清逼洪秀全封他为"万岁"，意图篡位。根本原因：农民阶级的局限性，无法克服小生产者的阶级局限性，无法建立科学的领导体制和权力制衡机制。

（3）影响：杨秀清被杀，韦昌辉被处死，石达开率部出走，太平军损失惨重，人心士气受到严重影响，太平天国由盛转衰。

（4）启示：①领导集团的团结是事业成功的关键；②要防止腐化堕落，保持艰苦奋斗的作风；③要建立科学的领导体制和民主监督机制；④农民阶级由于自身的局限性，无法单独领导革命走向胜利。
