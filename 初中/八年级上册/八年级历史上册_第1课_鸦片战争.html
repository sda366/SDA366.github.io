第1课 鸦片战争

八年级历史上册 · 第一单元 中国开始沦为半殖民地半封建社会

一、知识点总结（含时间线）
【单元时间线】
1839年

虎门销烟：林则徐在广州虎门海滩当众销毁收缴的鸦片，是中国人民禁烟斗争的伟大胜利。

1840年6月

鸦片战争爆发：英国舰队封锁广东珠江口，鸦片战争正式爆发。

1841年

三元里抗英：广州三元里乡民手持武器，包围英军盘踞的四方炮台，显示中国人民敢于反抗侵略的英雄气概。

英军再次北上，攻占厦门、定海、宁波。

1842年6月

英军进犯长江门户吴淞，江南提督陈化成力战牺牲。

1842年8月

《南京条约》签订：清政府被迫与英国签订中国近代史上第一个丧权辱国的不平等条约。

1843年

英国强迫清政府签订《虎门条约》，获得领事裁判权、片面最惠国待遇等特权。

1844年

清政府被迫签订中美《望厦条约》和中法《黄埔条约》，美、法获得各种特权。

【鸦片走私与林则徐禁烟】
背景知识
  • 19世纪30年代，西方已进入资本主义时代，英国成为头号工业强国
  • 英国为开辟海外市场，向中国运来呢绒、布匹，但遭到中国自给自足的小农经济的排斥，严重滞销
  • 中国向英国输出的茶叶、生丝等，销路旺盛
  • 为改变贸易不利局面，英国无耻地向中国大量走私毒品鸦片
鸦片走私的规模

从18世纪末到鸦片战争前夕，走私到中国的鸦片达40多万箱，从中国掠走3亿至4亿银元。

鸦片泛滥的严重危害
  1. 财政危机：白银大量外流，直接威胁到清政府的财政
  2. 政治腐败：许多官员吸食鸦片，导致政治腐败
  3. 军队削弱：士兵吸食鸦片，严重摧残体质，军队战斗力削弱
  4. 社会危机：鸦片毒害人民身心健康，造成严重的社会问题
林则徐禁烟
  • 上书皇帝：林则徐在给道光皇帝的上书中指出：如果听任鸦片泛滥，"是使数十年后，中原几无可以御敌之兵，且无可以充饷之银"
  • 赴广东禁烟：1838年底，道光皇帝派林则徐为钦差大臣，前往广东查禁鸦片
  • 收缴鸦片：林则徐到广州后，派人明察暗访，缉拿烟贩。英、美等国商贩被迫陆续缴出鸦片110多万千克
  • 虎门销烟：1839年6月3日至25日，在虎门海滩当众销毁收缴的鸦片
知识连贯性补充

前后联系：虎门销烟不仅是禁烟斗争的胜利，更成为英国发动鸦片战争的借口。英国以"保护贸易"为名，实则为了保护罪恶的鸦片走私利益，发动了侵略战争。

横向联系：同一时期，英国通过工业革命成为"世界工厂"，急需原料产地和商品市场，殖民扩张成为其国策。鸦片战争是英国殖民扩张政策在中国的体现。

历史启示：林则徐"苟利国家生死以，岂因祸福避趋之"的爱国精神，成为中华民族反抗外来侵略的精神象征。

【英国发动侵华战争】
战争爆发
  • 直接原因：中国禁烟的消息传到伦敦，英国政府公然支持罪恶的毒品走私，决定发动侵华战争
  • 时间：1840年6月
  • 标志：英国舰队封锁广东珠江口
战争进程
  1. 英军北上攻陷浙江定海，随后到达天津大沽口，威胁清廷
  2. 道光皇帝惊慌失措，将林则徐革职查办，改派琦善为钦差大臣前往谈判
  3. 1841年1月，英军攻占虎门附近的大角、沙角炮台，非法强占香港岛
  4. 道光皇帝下诏对英作战
  5. 英军进攻虎门炮台，广东水师提督关天培壮烈殉国
  6. 1841年秋，英军再次北上，攻占厦门、定海、宁波
  7. 1842年6月，英军进犯吴淞，江南提督陈化成力战牺牲
  8. 英军进攻镇江，守城清军全部牺牲
  9. 1842年8月，英军到达南京下关江面，清政府屈服

【《南京条约》等不平等条约的签订】
中英《南京条约》（1842年8月）

地位：中国近代史上第一个丧权辱国的不平等条约

主要内容：

  1. 开放广州、福州、厦门、宁波、上海五处为通商口岸
  2. 割香港岛给英国
  3. 赔偿英国2100万银元
  4. 英商进出口货物应纳税款，必须经过双方协议
中英《虎门条约》（1843年）

英国获得特权：

  1. 领事裁判权：英国侨民在中国犯罪或成为民事诉讼的被告时，不受中国法律管辖，只受本国领事依照本国法律审判的特权
  2. 片面最惠国待遇：清政府给予其他国家的任何权利，英国自动享有，但英国不给中国对等权利
  3. 在通商口岸租地建房等权利
中美《望厦条约》和中法《黄埔条约》（1844年）

美、法两国以武力相威胁，迫使清政府签订不平等条约。

主要特权：除享受英国在华取得的各种特权外，还扩大了其他侵略权益。

【鸦片战争的历史影响】
社会性质变化
  • 中国独立主权的完整性遭到破坏
  • 传统的小农经济逐步瓦解
  • 中国开始沦为半殖民地半封建社会
社会矛盾变化

此后，帝国主义和中华民族的矛盾、封建主义和人民大众的矛盾，成为近代中国社会的主要矛盾。

历史任务

反对帝国主义和封建主义，实现民族独立和人民解放，成为中国人民的历史任务。

历史地位

鸦片战争成为中国近代史的开端。

知识连贯性补充

与后续内容联系：鸦片战争后，中国开始沦为半殖民地半封建社会，这一社会性质决定了近代中国的主要矛盾和历史任务。此后的一系列历史事件，如第二次鸦片战争、太平天国运动等，都是在这个背景下发生的。

与单元主题联系：第一单元"中国开始沦为半殖民地半封建社会"的主题，正是以鸦片战争为开端的。从鸦片战争到第二次鸦片战争，中国半殖民地半封建化程度不断加深。

二、重点难点评析
第1课 鸦片战争
【深度评析】鸦片战争的深层原因

1. 根本原因——英国的殖民扩张需求

19世纪中期，英国完成工业革命，生产力飞速发展，急需原料产地和商品销售市场。中国地大物博、人口众多，成为英国觊觎的目标。在正常贸易中，中国处于出超地位（出口大于进口），英国为扭转贸易逆差，无耻地向中国走私鸦片。当禁烟运动触及其利益时，英国便以此为借口发动战争。

2. 直接原因——虎门销烟

1839年的虎门销烟，是中国人民正义的禁烟运动，却被英国侵略者当作发动战争的借口。这充分暴露了殖民主义者的强盗逻辑：他们可以任意倾销毒品，却不允许中国人民反抗。

3. 战争性质

鸦片战争是英国为维护罪恶的鸦片走私而发动的侵略战争，是非正义的。中国人民的禁烟运动和反抗侵略是正义的。

【广度评析】鸦片战争的多元影响

1. 政治层面

中国的领土完整和主权独立遭到破坏。香港岛被割占，五口通商，协定关税，领事裁判权等，使中国开始丧失独立自主的地位。

2. 经济层面

中国被迫开放市场，西方商品涌入，传统的小农经济开始解体。中国逐渐被卷入资本主义世界市场，成为西方列强的商品市场和原料产地。

3. 思想层面

战争惊醒了部分先进的中国人。林则徐编译《四洲志》，魏源提出"师夷之长技以制夷"，开启了近代中国向西方学习的先河。

4. 社会层面

社会性质发生根本变化，阶级关系出现新变化，民族矛盾上升为主要矛盾，反帝反封建成为近代中国革命的双重任务。

【高度评析】从鸦片战争看近代中国的发展道路

1. 落后就要挨打的历史教训

鸦片战争的惨败，深刻揭示了"落后就要挨打"的道理。清朝封建专制制度腐败，经济落后，军事装备陈旧，无法与完成工业革命的英国抗衡。这一教训成为近代中国追求现代化的重要动力。

2. 殖民主义的双重作用

马克思在论述殖民主义时指出，殖民主义具有双重作用：一方面是破坏性的，它摧毁了传统的社会结构；另一方面是建设性的，它客观上带来了新的生产方式和思想。鸦片战争虽然给中国带来深重灾难，但也在客观上促使中国走向近代化。

3. 近代中国的历史主题

鸦片战争后，实现中华民族伟大复兴成为中国人民的历史主题。这一主题贯穿整个中国近代史，也是理解此后太平天国运动、洋务运动、戊戌变法、辛亥革命等历史事件的关键线索。

4. 世界历史视野下的鸦片战争

从世界历史的角度看，鸦片战争是西方工业文明与东方农业文明碰撞的缩影，是资本主义全球扩张的必然结果。它标志着中国被动地卷入世界现代化进程，开启了中国近代化的艰难历程。

三、教材栏目解答
第1课 鸦片战争
【想一想】为什么鸦片战争以中国失败而结束？

答案：

鸦片战争以中国失败而结束，主要有以下原因：

  1. 制度腐败：清朝封建专制制度腐败，统治者昏庸愚昧，对内敌视人民，对外妥协投降
  2. 经济落后：中国仍是自给自足的小农经济，而英国已完成工业革命，经济实力强大
  3. 军事落后：中国军队使用刀、矛、弓箭和旧式的抬枪、火炮，抵挡不住英军新式的步枪和大炮
  4. 指挥失当：清政府战和不定，将帅无能，多次临阵换将，动摇军心
  5. 对外妥协：统治阶级中投降派占据上风，最终向侵略者屈服

根本原因：腐朽的封建制度无法对抗先进的资本主义制度。

【材料研读】
马克思引用英国人蒙哥马利·马丁的话说：

"奴隶贸易"比起"鸦片贸易"来，都要算是仁慈的。我们没有毁灭非洲人的肉体……没有败坏他们的品格、腐蚀他们的思想，也没有毁灭他们的灵魂。可是鸦片贩子在腐蚀、败坏和毁灭了不幸的罪人的精神存在以后，还杀害他们的肉体……

——[德]马克思《鸦片贸易史》

阅读材料，谈谈鸦片走私带来的严重危害。

答案：

马克思的这段话深刻揭露了鸦片贸易的罪恶本质：

  1. 摧残身体：鸦片严重摧残吸食者的身体健康，使人精神萎靡、体质虚弱，甚至死亡
  2. 腐蚀灵魂：鸦片不仅毁灭肉体，更败坏人的品格、腐蚀思想、毁灭灵魂
  3. 危害家庭：吸食鸦片导致家庭破裂、妻离子散
  4. 败坏社会风气：鸦片泛滥导致社会道德沦丧，赌博、偷盗等犯罪增多
  5. 削弱国力：官员、士兵吸食鸦片，导致政治腐败、军队战斗力下降
  6. 经济崩溃：白银大量外流，国库空虚，财政危机严重

鸦片贸易比奴隶贸易更为罪恶，因为它不仅剥夺人的生命，更从精神层面摧毁人。

（1）根据材料一，分析英国发动鸦片战争的根本原因。

（2）材料二反映了鸦片战争的什么特点？

（3）根据材料三，概括鸦片战争对中国社会的影响。

（4）结合所学知识，谈谈鸦片战争的失败给近代中国带来了什么启示？

参考答案：

（1）英国完成工业革命后，急需原料产地和商品销售市场。为开辟海外市场，向中国大量走私鸦片，当禁烟运动触及其利益时，便以此为借口发动侵略战争。

（2）反映了英国侵略的蓄谋已久和有计划性。英军从广东北上，攻陷定海，到达天津，体现了其战略进攻的特点。

（3）鸦片战争改变了中国历史发展的进程：①中国独立主权的完整性遭到破坏；②传统的小农经济逐步瓦解；③中国开始沦为半殖民地半封建社会；④成为中国近代史的开端。

（4）启示：①落后就要挨打，必须实现国家富强；②腐朽的封建制度无法对抗先进的资本主义制度，必须进行制度变革；③要勇于学习外国的先进技术和思想；④要坚决反抗外来侵略，维护国家主权和领土完整。

材料：马克思在《鸦片贸易史》中指出："奴隶贸易"比起"鸦片贸易"来，都要算是仁慈的。我们没有毁灭非洲人的肉体……没有败坏他们的品格、腐蚀他们的思想，也没有毁灭他们的灵魂。可是鸦片贩子在腐蚀、败坏和毁灭了不幸的罪人的精神存在以后，还杀害他们的肉体……
请回答：

（1）根据材料，概括鸦片走私带来的危害。

（2）结合所学知识，分析清政府为什么要派林则徐前往广东查禁鸦片？

（3）虎门销烟有什么重要意义？

参考答案：

（1）鸦片走私带来的危害：①摧残吸食者的肉体健康；②败坏人的品格、腐蚀思想、毁灭灵魂；③导致白银大量外流，威胁清政府财政；④导致政治腐败和军队战斗力削弱；⑤造成严重的社会问题。

（2）原因：①鸦片泛滥给中华民族带来深重灾难；②白银大量外流直接威胁到清政府的财政；③许多官员、士兵吸食鸦片，严重摧残体质，导致政治腐败和军队战斗力削弱；④林则徐力主禁烟，认识到鸦片泛滥将危及清朝统治。

（3）虎门销烟是中国人民禁烟斗争的伟大胜利，显示了中华民族反抗外来侵略的坚强意志。林则徐的爱国精神和民族气节，成为中华民族的宝贵精神财富。
